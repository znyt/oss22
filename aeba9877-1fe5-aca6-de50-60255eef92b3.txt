　　叶少阳继续捏着这张引魂符，陷入了沉思之中，直到秋镇长轻声在旁边叫他，才醒过神来。
<br/>　　“叶先生，事情……结束了吗？”
<br/>　　“结束了。”叶少阳瞥了他跟阿兰一眼，道：“我也该走了。”
<br/>　　阿兰感激涕零，各种道谢。
<br/>　　秋镇长让老妈子扶阿兰先进屋，自己送叶少阳来到偏厅。
<br/>　　翠云早己等得不耐烦，把糕点和糖都快吃完了，见叶少阳进来，急忙一抹嘴，站起来，上下打量了他一会，道：“你没事吧。”
<br/>　　“没事，可以回家了。”
<br/>　　秋镇长道：“今天很晚了，叶先生也累了，不然在我这住一晚，明天我再设宴款待如何？”
<br/>　　叶少阳坚持要回去，并且不让秋镇长送，跟着翠云一起回去了。
<br/>　　路上，叶少阳把大致经过讲了一遍，翠云又是感慨，又是为他紧张，更有点埋怨他把自己丢在房间里，没有看到与鬼打架的一幕。
<br/>　　回家之后，翠云让叶少阳先去休息，自己去烧热水给他洗澡。叶少阳回到房间里，就着煤油灯，仔细检查了自己那张引魂符，确定没有任何问题，对着灯火发了一会呆，叹道：“我早就该想到的……”
<br/>　　没错，引魂符不能用，不是别的原因，唯一的原因只能是：自己对于这个世界，是个外来者！
<br/>　　凡是人间法师，只要是道童以上牌位，在阴司里都是有记录的，到了天师牌位，还要去阴司挂玄名，阴司这么做，说的直白一点，是为了给这些人间法师定位，这样一旦有法师邪修之类，阴司虽然不能派人来追杀，当人间法术界清理门户的时候，还可以提供一些帮助。
<br/>　　还有，这也是阴司笼络人间法师的一种手段：认可你的地位，给你提供一些法术的资源，这样你死了之后，总不好拒绝阴司对你的安排和指挥了吧？
<br/>　　一切法术，法力都有源泉。
<br/>　　一切跟阴司有关系的法术，法力的源泉，都来自阴司。
<br/>　　叶少阳是个外来者，没有在这个世界的地府挂玄名，因此施展的一切与阴司有关的法术，得不到法力的来远，因此送鬼魂去阴司的引魂符，才没有任何灵力的倾注，所以失败了。之前那些法术能够成功，因为这些法术的源泉，并非来自阴司，而是来自天地律法，说白了就是天真地秀日精月华的力量。
<br/>　　叶少阳既然身在这个世界，那么就与这个世界的任何人一样，可以用法术调动一切天地间的力量，除了有关阴司的。
<br/>　　为了验证自己的猜测，叶少阳解下了缠在腰上的勾魂索，倾注罡气，勾魂索立刻通灵，但是当叶少阳尝试运转兜率八卦鞭，却没有引来地狱的力量。
<br/>　　果然……自己没有猜错，叶少阳用力喘了一口气，勾魂索在人间，可以吸收天地灵气，但是兜率八卦鞭是地府的法术，依靠的也是阴司提供的灵力，自己没有资格调动，自然就失败了。
<br/>　　这个结果并没有让叶少阳感到沮丧，反正自己大部分法术都跟阴司没有关系，最多也就是没法用勾魂索，没法画引魂符之类，与之相反，这个发现，反而引起了叶少阳对这个世界的重新思考。
<br/>　　不光是道渊真人，在这个世界，自己其实还有很多熟人：
<br/>　　崔府君，牛头马面，黑白无常，钟馗天师，徐文长……还有自己在阴司的好哥们萧逸云，在这个世界的阴司，他们一定也是存在的！
<br/>　　那么，这就产生了一个悖论：同样的人，或者说是生灵，怎么可能有两个，怎么可能出现在两个世界？如果用穿越时空的理论，勉强解释的话，叶少阳也能够说服自己，但是，一个最最重要的问题浮了上来：自己在自己那个世界的熟人，在这个世界，认识自己吗？
<br/>　　说起来应该是不认识，因为这是民国时期，按道理自己还没出世，不该出现在这个世界，但自己却偏偏真就出现了……
<br/>　　这时候翠云推门进来，告诉叶少阳洗澡水烧好了，叶少阳脑子里正在思考这些，浑浑噩噩地跟着她走，进了厨房，看见有个热气腾腾的木桶，于是脱了衣服……
<br/>　　“哎呀！”
<br/>　　翠云一声惊叫，将叶少阳从思绪中拉出来，转头看去，翠云背过身，捂住了眼睛。
<br/>　　“少阳你怎么也不等我走了再脱衣服！”
<br/>　　叶少阳低头看自己，已经光溜溜的了……顿时窘迫的不行，急忙跳到热水桶里，结果嗷的一声跳了出来。
<br/>　　“哎呀，我还没加冷水呢，虽然不是开水，但也够热的，你怎么就跳下去了！你没烫坏吧！”
<br/>　　翠云情急中转身，又看到了少儿不宜的画面，赶紧又捂着脸。
<br/>　　叶少阳本来烫的直跳，见到这一幕，急忙护住自己重点部位，一时间不知道怎么办，简直狼狈死了。
<br/>　　翠云跑了出去，把门从外面关上，说道：“缸里有热水，你自己加水吧，我去睡了……”
<br/>　　叶少阳洗完澡，回到自己房间里，把门拴好，免得翠云过来打扰，看见自己一动不动，还不知道吓成什么样子。
<br/>　　他上床之后，把被子扔开，在床上布置了一个法坛，然后作法走阴……他要去阴司见自己那些熟人，即使他们都不认识自己，自己也要把情况告诉他们，至少要告诉崔府君！
<br/>　　就算这个崔府君，不是自己世界的那个，可崔府君是阴司最有能耐的人，如果自己把一切说出去，他或许有办法送自己回去？
<br/>　　至少，他一定能给自己一些帮助。
<br/>　　然而，想法是好的，现实却是残酷的：走阴需要的法力，也是来自阴司……换句话说，没有阴司的授权，自己连走阴的资格也没有。
<br/>　　叶少阳尝试了几次都失败了，最后无力地倒在床上。巨大的孤独感和无力感包围了他。他从背包里拿出自己的钱包，除了一身法器，这是自己从那个世界带来的唯一的东西了，里面有他跟芮冷玉拍的婚纱照，做成卡片的形势，那是在空界之战爆发之前，他跟芮冷玉去影楼拿来的。当时就爱不释手，塞进了钱包里，没想到现在却成了对那个世界唯一的念想。
