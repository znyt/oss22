　　第710章
<br/>　　对于崔白玉的心路历程，程大雷未必不知，只是没有去关心。
<br/>　　因为这种事程大雷实在见得太多。每天都有大量流民迁入凉州，几乎每人背后都有一部苦难史。妻离子散，亡国亡民，如果程大雷每个都关心，实在有些忙不过来。
<br/>　　在这些人中，崔白玉的经历实在算不上特殊，这样一个糟糕的世道，没理由你崔白玉可以幸免于难。
<br/>　　长安城那边乱成一锅粥，隔几天就有最新的消息传来。相比之下，凉州城这边却渐渐步入正轨，程大雷的工作也渐渐轻松起来。他还可以抽空和苏樱在街上逛逛。
<br/>　　这一日，程大雷和苏樱相伴出了凉州城。经过几次相处，二人间的心结差不多已被撕扯开，程大雷与苏樱相处，也不再觉得有什么压力。
<br/>　　阳光正好，野草闲花杂乱铺在草原上。二人信马由缰，也没有固定的方向，只是随便出来散散心。
<br/>　　秋夏之交，天气仍有几分炎热，有老农在田间劳作，几个稚童在地头撒野。
<br/>　　程大雷和苏樱下了马，在田野上漫步，都是养熟的战马，随意放它们去吃草，也不担心跑丢。
<br/>　　“依你看来，李乐天能稳住长安城的局势么？”苏樱随口问道。
<br/>　　“按现在的情况看来，他现在不至于出什么大乱子。但也要看怎么说。”
<br/>　　“怎么讲？”苏樱不解道。
<br/>　　二人在路旁一棵大树下歇了，从马背上拿出水食。程大雷道：“李乐天在军中是有威望的，当初困守长安城，他出了很大力气。虽然后来给他按个守城不力的罪名，但他究竟做过什么事，军中还是有许多人看在眼里的。没有人心，他这次行动也不可能成功。不过……”
<br/>　　“不过什么？”
<br/>　　程大雷苦笑着摇摇头：“帝国之灾，起于流民，但根源还是在各方豪族。豪族势大，朝堂之上太多尸位素餐，庸庸碌碌之徒。普通人只因为出身不好，便难以被重用。李乐天开始提拔军中的平民子弟，就是看出了这一点。不过，他拔掉了相府这颗毒瘤，同时也需要豪族的支持。这块蛋糕太大了，李乐天想要一个人吞下，却没有那么大的肚子，只能是把蛋糕分出一些给各方豪族。”
<br/>　　苏樱颦起眉头：“你说得我大概都懂，只不过蛋糕是什么？”
<br/>　　“呃……哈哈。”程大雷干笑一阵：“道理大概就是这么个道理，李乐天取代相府，执掌整个京州已经是大势所趋。但是，长安城不管是姓崔，还是姓李，其实都没多大区别。”
<br/>　　程大雷咕咚咚喝了一口水，心下也有些感慨，道：“曾有人问过我，帝国还有救么。如一栋老房子，漏雨了补瓦，漏风了补墙，补来补去，打满了补丁。索性不如一把火烧掉，在原地重建。”
<br/>　　苏樱也并不只是会在房中缝缝补补，与程大雷生些闷气，事实上，她比程大雷还要更聪明几分，只是没机会显露本事而已。
<br/>　　她沉默思索着程大雷的话，半晌开口道：“各路诸侯争名夺利，争来争去，却又为了那般，将天下搞成这个样子。”
<br/>　　程大雷道：“争名夺利，关键不在名利，而在一个争字。帝国之大，名又如何，利又如何，一个人天大的肚子，一天也不过三顿饭。所以，名利并非他们追逐的目的，他们在意的只是争夺本身。”
<br/>　　苏樱一脸困惑的看着程大雷，不懂他话中的意思。
<br/>　　程大雷道：“你是女儿家，不懂男儿的心思。大丈夫生于世，总是要去做些事的。女人、财富、地位这些东西不过是过往云烟，见风则散。这些东西可以满足一部分人，但不可能满足所有人。所以读书人说为天地立心，为生民立命，为万事开太平……”
<br/>　　“话是这么说，但归根结底想说的只有一句话：我既然来这世间，不能白白走一遭。有这七尺之躯，总是要做一些事的。如今，好不容易逢着这么个乱世，天下英雄会武。有几个人肯甘于寂寞，谁不想着施展拳脚，和天下英雄碰一碰。我要赢，我要赢过天下人，毕竟赢这个字眼，对男人来说的确太重要了。”
<br/>　　苏樱依旧颦着眉头，叹口气道：“谁懂你们男人的心思，偏偏又是你们把帝国搞成这个样子。偏偏还要打着天下苍生的名义。”
<br/>　　程大雷点点头：“都说为了天下苍生，可那家诸侯又真的会把天下苍生放在眼里，不过是为了自己的利益，打着正义的旗号。有时候同样的话说得多了，连自己都要信以为真。”
<br/>　　“那么，我们不与人争不行么？”苏樱问。
<br/>　　“不行。”程大雷摇摇头：“既然上了这牌桌，就没有轻易下场的道理。我们现在的日子多少还过得去，可这样的日子，还不是一路杀出来的，等那一日我提不动斧头了，太平日子也就结束了。”
<br/>　　苏樱依偎在程大雷怀里，男人要做的事，女人有时候真无法明白，不过，多多少少也能理解他肩上的压力。
<br/>　　二人目光正好看向同一处，几个幼童本来玩得开心，但为一只蚂蚱突然打了起来。最后，一个孩子被打哭，哇呀呀抹着眼泪。高个的孩子握着蚂蚱，恶狠狠道：“我的，是我的。”
<br/>　　蚂蚱已被他手掌掐死，但他仍紧紧攥着，不肯松手。
<br/>　　程大雷静静看着这一幕，慢悠悠道：“世间哪有什么太平日子，便是黄口幼童也要争个强弱，何况我们。”
<br/>　　正这时候，一匹快马从城中奔来。马上一名小校，在来到程大雷身边时，翻身下马，双膝跪倒在地。
<br/>　　“启禀大当家，徐军师请大当家回城，说是有要事相商。”
<br/>　　程大雷叹了口气，道：“难得浮生半日闲呐，说是什么事了没有？”
<br/>　　那小校道：“似乎是武威、浩甲、金城三地来人，具体事情小人也不知。”
<br/>　　程大雷点点头，树倒猢狲散，崔相的大树倒了，看来这三家也要找新的大树了。
<br/>　　他翻身上马，与苏樱两骑并一骑，快马返回凉州城。
<br/>(本章完)
