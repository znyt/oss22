        &lt;h3&gt;第三卷第二百六十一章  血缘之缘&lt;/h3&gt;
<br/>
<br/>        陈子昂一行几个人静静的站在门口，看着病房里俩个老人低低私语，都不忍心打破病房的温馨。
<br/>
<br/>        那甜蜜的语气就像是热恋中的人，那满满的爱意浓得像陈年的酒，让闻者沉醉。
<br/>
<br/>        陈元良侧脸看到了陈子昂一行几人，拍了拍冉灵卉：“子昂回来了。”
<br/>
<br/>        冉灵卉转过身：“子昂，你回来了？这几位是？来！快来坐。”
<br/>
<br/>        “爸，妈。”陈子昂的声音有些哽咽，这么多年一直在外面打拼，何曾想过父母亲会老去，会生病？树欲静而风不止，子欲养而亲不待。
<br/>
<br/>        “子昂啊，这是来客人了？”陈爸爸手上挂着点滴，单身撑着准备坐起来，陈妈妈立马转身扶着陈爸爸坐起来。
<br/>
<br/>        “大哥。你是元良大哥吗？”代凉忐忑的走到陈爸爸的面前，伸出手，握着了陈元良的手。她的手微微颤抖着，她只知道她的哥哥叫“陈元良”，和她长得一样的瓜子脸和丹凤眼。
<br/>
<br/>        “你，你是凉儿？”陈元良的声音颤抖着迟疑着，惊喜着。
<br/>
<br/>        “我，我是代凉，我是凉儿。你看，这个是手镯。”代凉抓过安男的手，取下安男手腕上的手镯。
<br/>
<br/>        她感觉眼前的清瘦的老人就是她的哥哥，她找了他这么多年。
<br/>
<br/>        陈子昂和陈子建惊呆了。
<br/>
<br/>        “凉儿，哥哥找得你好苦啊。”陈元良一把拔下了手中的点滴，拥抱着代凉，老泪纵横。
<br/>
<br/>        人生无常，有些人一分别就是一生，有些人用一生去寻找，寻找爱情，寻找亲情，沧海桑田，天涯路远，他们有幸还能遇见。
<br/>
<br/>        安男拉着安林曦的手，丹凤眼里储满了泪水，还好，能让母亲找到亲人，还好，他们来得及时。
<br/>
<br/>        冉灵卉看着陈元良和代凉相认，心里百感交集，老太爷啊，真的算是开眼了，来完成陈元良的心愿。
<br/>
<br/>        冉灵卉稳了稳神：“元良，快让妹妹坐下，坐下说，你也回到床上去，子霖，去找护士来给你爸爸重新挂上水，子昂，来招呼大家坐。”
<br/>
<br/>        陈子昂和陈子霖像儿时一样听着妈妈的吩咐各自从震惊中回过神来。
<br/>
<br/>        “安教授，安男快请坐。”陈子昂的声音也透着喜悦，她从来都没有想到过安男和她是亲戚，只是觉得和安男在一起很亲切，原来代凉就是爸爸失散多年的妹妹，难怪感觉和她说话有一种莫名的亲近感。
<br/>
<br/>        陈子霖也叫来了护士，重新给陈爸爸挂上了点滴。
<br/>
<br/>        “说曹操，曹操到，我们俩人刚刚还在说你们，你们就出现了，上天真的是很好，很公平。”冉灵卉笑着说。
<br/>
<br/>        “是啊，从我知道我有了哥哥之后，我一直都没有放弃找你们，一直找了这么多年，我终于是找到你们了，我有哥哥了。”代凉心情激动，她一直站在陈元良的病床前，不肯就坐。
<br/>
<br/>        “凉儿，坐下说，坐下说。”陈元良拍了拍床沿。
<br/>
<br/>        “好。”代凉轻轻的坐在病床前。
<br/>
<br/>        “你告诉哥哥，你这么多年是怎么过来的？”陈元良急切的问，他的脸上泛着潮红。
<br/>
<br/>        “我是听爸爸妈妈告诉我的，说我并不是他们亲生的，我还有父母亲和一个哥哥，让我有机会了找一找，他说原本两家关系很好，只是家里发生了变故，你们不方便带我走，就将我留在了他们家里，只是后来他们也几经搬迁，离开了原来的地方，我现在姓代，也是代家唯一的女儿，爸爸妈妈对我很好，所以我才有现在的生活，这个是我的爱人，安林曦，这个是我的女儿安男。”
<br/>
<br/>        “代叔叔他们好吗？”陈元良问。
<br/>
<br/>        “他们都不在了，现在只留下我一人。”代凉提起养父母心里有些伤感，可是找到了哥哥让她又心生喜悦。
<br/>
<br/>        “我一直在找你，我回老家了几次，都找不到代叔叔去了哪里，然后托子建也到处找你.......”
<br/>
<br/>        陈元良和代凉俩人你一言我一语的说开了，完全忘记了病房还坐着其他的人。
<br/>
<br/>        “子昂，我给他们都打电话，让他们都过来，晚上一起吃顿团圆饭。”陈子霖轻声的和陈子昂商量着。
<br/>
<br/>        “行，姑姑回来了，也该让大家见见。”陈子昂直接就改了口。
<br/>
<br/>        “那我出去打电话，你陪着姑父和妹妹坐着。”陈子霖可以告诉二哥，找到姑姑了，二哥也不用那么内疚了。
<br/>
<br/>        陈子建从冯睿哪里回来之后，一直泱泱不乐，但是又无处诉说，他的心疼得不能呼吸，没有想到和他同床共枕的几十年的人，心思如此深沉，那之前的恩爱她是如何能装得出来？她怎么能这样对待他的亲人？
<br/>
<br/>        俗话说，爱屋及乌啊！
<br/>
<br/>        陈子建感觉自己生了一场大病，他浑身发软，四肢无力，两眼昏花，可是看着躺在病床上的老父亲，他更是心生愧疚，那种蚀骨的悔恨让他四肢百骸都蔓延着疼楚，但他还不能将冯睿怎么样。
<br/>
<br/>        弟弟妹妹们都很忙，就他如今闲赋在家，照顾老父亲的担子就轮他挑起来了，父亲只能吃一点流食，过几天要做手术，他现在到陈子霖的家里给父亲带饭。
<br/>
<br/>        “慧芝，这几天辛苦你了，要做几样的饭菜。”陈子建对严慧芝说。
<br/>
<br/>        严慧芝虽然没有读多少书，但是把家里操持得很好，尤其是把几个孩子教育得很好，一个个温文有礼，进退有度，比他家伟冯要好很多，现在想一想真是庆幸没有让天天跟着他们一起生活。
<br/>
<br/>        “有什么辛苦的，伺候老人和孩子就是我的责任，我除了能照顾他们，我还能做什么？”严慧芝笑着说。
<br/>
<br/>        “天天最近好吧？”陈子建对女儿天天也心生愧疚，一生下来就交给奶奶带，后来又是严慧芝带，他们都不曾管过她。
<br/>
<br/>        “你看她像不好的人？天天没心没肺的，快乐的就像是一只小鸟。”严慧芝语气宠溺。
<br/>
<br/>        “妈咪，你家天天天天回来啦，我好饿啊!”说曹操曹操到，陈天天如一只快乐的小鸟从外面飞了回来。
<br/>
<br/>        “二爸，你又来啦？饿不饿？吃没吃饭？咦，咋了？一副萎靡不振的样子？这是失恋的节奏？”陈天天噼里啪啦酱。
<br/>
<br/>        “你这孩子，怎么这样和你二爸说话，没一点样子。”严慧芝笑着责怪陈天天。
<br/>
<br/>        “啵。”严慧芝脸上狠狠的被陈天天啄了一口。
<br/>
<br/>        “嗯，真香。”陈天天砸吧着嘴对严慧芝说：“妈咪，你家天天今天不去上学了，想休息半天，你给个话，我准备去和我们家的床谈一场梦寐已久的恋爱。”
<br/>
<br/>        “你都被床黏住了，在这样下去，连毕业证都拿不到。”严慧芝捏了一下陈天天有点婴儿肥的小脸。
<br/>
<br/>        “嘻嘻，你觉得陈家人会拿不到毕业证？你这不是在打二爸的脸吗？是不是啊二爸？”陈天天调皮的一笑，开始求助陈子建了。
<br/>
<br/>        陈子建正准备说话被电话铃声打断了。
<br/>
<br/>        “你说什么？找到姑姑了，现在在病房？晚上一起吃团员饭？好好好！”陈子建挂了电话对严慧芝说：“慧芝，晚上不要做饭了，带着孩子我们一起出去吃，子霖说找到姑姑了，要吃团员饭。我来给大姐四妹五妹打电话。”
<br/>
<br/>        陈子建一个一个的打着电话，将喜悦传递给自己的姐妹们，邀请大家一起聚聚。
<br/>
<br/>        最后他拿起电话，想了又想，给江俞轩拨打了一个电话：“俞轩，你今晚忙不忙？”
<br/>
<br/>        “子建哥，好久不见，我不太忙，你有什么吩咐？”江俞轩刚好有事情要找陈子建。
<br/>
<br/>        “不忙晚上就一起吃饭，地址在大唐相府。不见不散，我们家今天有喜事！”陈子建喜滋滋的挂了电话，江俞轩曾经让他看手镯呢，他肯定想不到，他的姑姑找到了。
<br/>
<br/>        “二爸，谁的姑姑？”陈天天好奇的问陈子建。
<br/>
<br/>        “我的姑姑啊。”
<br/>
<br/>        “咦，了不起原来你也有姑姑。”
