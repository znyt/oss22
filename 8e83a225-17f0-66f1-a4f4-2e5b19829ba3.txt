　　叶尘不想恃强凌弱，更不想搞特殊化，既然是自己有错在先，那就应该接受惩罚，而且一千俯卧撑，对他来说，简直轻而易举。
<br/>　　而其他学生，先是对王教官主动向叶尘道歉的举动，感到深深的诧异，听到叶尘主动要求做一千个俯卧撑，顿时纷纷再次嘲笑起来，
<br/>　　“这家伙还真能口出狂言啊！”
<br/>　　“就算是特种兵，能一口气做五百个的也不多吧！”
<br/>　　“一千个俯卧撑，他是准备做到明天吗？”
<br/>　　……
<br/>　　然而，
<br/>　　随着叶尘双手撑在地上，开始正式做起了俯卧撑，所有人顿时全看傻眼了。
<br/>　　只见，
<br/>　　叶尘不但做俯卧撑的动作标准无比，而且速度奇快，几乎是一秒钟做三个的高频率！
<br/>　　“握草！好快！都已经三十个了！”
<br/>　　“没想到这家伙，看上去柔柔弱弱的，爆发力竟然这么强！”
<br/>　　众人惊叹过后，很快又有人出言嘲讽起来，
<br/>　　“哼！照他这个速度，最多一百个，就肯定爬不起来了！”
<br/>　　“做俯卧撑最重要的就是要把握好节奏，一组组的做，才能更持久，哪有像他这样的！”
<br/>　　“我敢打赌，他最多超不过70个！”
<br/>　　……
<br/>　　然而，
<br/>　　叶尘速度不但没有丝毫减弱的迹象，反而越来越快，一转眼的功夫，就已经超过了70个，然后又很快突破了100个！
<br/>　　200个……
<br/>　　300个……
<br/>　　才短短两分钟，叶尘已经做了三百多个俯卧撑。
<br/>　　之前不看好叶尘的那些人，顿时一个个全都闭上了嘴巴。
<br/>　　就连一旁的王教官，也顿时两眼反光，脸上露出一抹惊愕之色。
<br/>　　之前张上校并没有告知他，叶尘的真实身份，在他想来，这家伙多半是某个权贵子弟，所以才能清楚自己的顶头上司。
<br/>　　因而，王教官虽然碍于张上校的命令，不敢多说什么，可是心里终归有些不爽。
<br/>　　如今，看到叶尘的表现，他之前心头的那一丝不快，顿时便消失的无影无踪了。
<br/>　　军中之人都崇尚强者，王教官自然也不例外，就叶尘眼前的表现来看，已经超越了那些特种兵，他更是远远比不上！
<br/>　　然而，让王教官更加感到震惊的是，叶尘丝毫没有要停下来的意思，而且做俯卧撑的速度，也丝毫没有减慢分毫！
<br/>　　“已经突破500个！”
<br/>　　不知道谁惊呼了一声，众人望向叶尘的表情，也开始发生细微的变化，从一开始的不屑，到吃惊，到震惊，再到难以置信……
<br/>　　很多人，像看怪物一样，看着那道依旧在上下起伏单薄身影，已经没有人，再敢出言嘲讽。
<br/>　　“600！”
<br/>　　“700！”
<br/>　　……
<br/>　　叶尘以每分钟将近二百个俯卧撑的速度，只用短短三四分钟，就已经突破700个。
<br/>　　照这种速度下去，完全一千个，似乎也并非完全不可能。
<br/>　　“我的天呐！他竟然……一口气做了700多个俯卧撑！”
<br/>　　“难道他真的打算，一口气做完一千个吗？”
<br/>　　“这还是人吗？难道他是机器人不成？”
<br/>　　“太变态了！”
<br/>　　……
<br/>　　就连身为内劲武者的唐威，看到这一幕，眼眸中也不由得闪烁着一丝精芒，心中暗暗寻思起来，
<br/>　　“难怪我一直觉得他不简单，果然不是我的错觉！只是，他如果也是内劲武者的话，为何我竟然在他身上，察觉不到一丝气劲？难道他已经达到返璞归真的境界了吗？”
<br/>　　杨之凡的脸上，也露出一抹凝重之色，
<br/>　　‘看来是我小瞧他了，回头必须让人好好调查一下此人的背景！’
<br/>　　“908，909，910……”
<br/>　　全场最兴奋就要数李虎了，这家伙此时满脸激动的报着数，整个人已经忍不住手足舞蹈起来。
<br/>　　终于，
<br/>　　“1000！”
<br/>　　随着李虎报出最后一个数字，叶尘双手在地上微微一撑，整个人已经稳稳的站立起来，而且脸色平静如水，脸不红，心不跳，缓缓走回队列之中。
<br/>　　众人顿时再次相顾骇然，
<br/>　　看叶尘这架势，一口气做完了1000个俯卧撑，竟然连大气都不喘一口，仿佛只是做了一件微不足道的小事！
<br/>　　“居然只用短短五分钟！这体力也太好了吧？”
<br/>　　“看他的样子，好像再做一千个，也不是不可能！”
<br/>　　“简直不是人啊！”
<br/>　　“我忽然有点明白，他为什么可以讨得那位萧女神的欢心了！”
<br/>　　……
<br/>　　众人纷纷由衷的感叹起来。
<br/>　　经过这一段小风波，已经没有人再敢轻视叶尘，就连王教官也发自内心的，对叶尘肃然起敬。
<br/>　　“叶尘同学，你需要休息一下吗？”
<br/>　　王教官热心的开口询问道。
<br/>　　叶尘淡淡一笑，
<br/>　　“不用！”
<br/>　　王教官又深深的看了叶尘一眼，这才将目光挪开，
<br/>　　“好了，全体都有！军训继续！”

<br/>　　为时两周的军训生活，总算是结束了。
<br/>　　而与此同时，对于大一新生来说，真正的大学生涯，也正式开始！
<br/>　　自从叶尘一鸣惊人之后，整个生物学院的新生，对叶尘刮目相看的同时，还生出一丝惧意，已经不敢再随便议论他。

<br/>　　这种平凡大学生活，并没有维持太久。

<br/>　　叶尘趁着周末的时间，带曦月到郊外的游览区游玩，没想到路上竟然被人跟踪！

<br/>　　叶尘故意带着曦月，来到一处偏僻的树林之中。

<br/>　　“是啊！你要是当初把他杀了，不就什么事都没了？要怪，只能怪你自己太蠢！”
<br/>　　其余几人也纷纷大笑起来，如同看猎物一般望着叶尘。

<br/>　　“那你可知道，我当初为何不杀他吗？”
<br/>　　“为何？”
<br/>　　叶尘两眼微微眯起，
<br/>　　“因为在我的眼中，他不过就是一只，可以随时碾死的蝼蚁！”
<br/>　　“你们，也是一样！”
