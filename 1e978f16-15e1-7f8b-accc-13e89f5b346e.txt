　　谁也不知道，穷奇为什么能够发现她，并且猝不及防地将她掉一口叼住了她，现出真身的秋莹剧烈挣扎，但一动不能动。
<br/>　　“秋莹！”
<br/>　　叶少阳大吼起来，立刻要冲出去，冲和子一步冲过来，死死拦住他，“叶天师，眼看法阵将成，你若出去，阵法便破了，前功尽弃啊！”
<br/>　　“说什么，上啊！”小马也喊了一声，招呼大家一起上。
<br/>　　“等下！”
<br/>　　叶少阳反而冷静下来，大喝了一声，一时间，所有人都看着他。
<br/>　　秋莹被穷奇叼在口中，但还没有吞下去，显然是想用她来做人质。
<br/>　　叶少阳深吸一口气，朝赤月罗刹等人看去。
<br/>　　赤月罗刹眉飞色舞，冲叶少阳笑道：“怎么，叶少阳，你怕了吗，你手下门人这么多，死个把人，又有什么关系。”
<br/>　　“怎样才能放人？”叶少阳不想跟她啰嗦，开门见山地问道。
<br/>　　赤月罗刹想了一下，道：“打开结界，不许防守。”
<br/>　　这是一个他们无法接受的条件，结界一旦打开，一切都完了，之前努力的一切也都白干了。
<br/>　　“玉面罗刹在我手中，我们交换人质！”小马拍了下玉面罗刹的屁股，吼道。
<br/>　　赤月罗刹看了一眼小马怀中的玉面罗刹，咬了咬牙，说道：“我们太阴山的人命不值钱，一旦落入敌人手中，就当是死了。”
<br/>　　小马大声骂起来。
<br/>　　大伙互相张望着，一个个神情复杂，最后都朝叶少阳看过去。
<br/>　　“叶掌教，成大事则不拘小节，节哀啊……”慕寒在远处淡淡劝道，语气中带着一丝嘲讽的意味。
<br/>　　“草泥马，你是哪根葱！”
<br/>　　小马指着他骂起来，但这当口，谁也没有工夫去跟他磨蹭。
<br/>　　“少阳，你怎么做，我们都听你的！”
<br/>　　小九握着叶少阳的胳膊，殷切地望着他。
<br/>　　“叶天师，大局为重……”冲和子劝着。
<br/>　　“叶天师，我劝你快一点，徐文长点化灵石，顷刻之间就要完成，若到那时，你这门人却是必死无疑了！”赤月罗刹面带微笑，似乎已经猜到叶少阳会如何选择了。
<br/>　　穷奇嘴里叼着秋莹，落在结界的正前方，它显然是有智商的，并不急着杀死秋莹。
<br/>　　“穷奇！！”
<br/>　　叶少阳望着穷奇，大声喝道，“你是上古四大凶兽之一，就干这种勾当？你要是条汉子，就丢下她，我这就跟你打，说到做到！”
<br/>　　穷奇望着他，浑浊的目光之中，透着一丝犹豫之色。
<br/>　　“你们无论如何守好这里，这是命令！”叶少阳咬牙对大伙说道。
<br/>　　大伙当然不同意，要跟他一起出去。
<br/>　　叶少阳反手推开他们，从佛光结界中走了出去。
<br/>　　穷奇望着他，嘴巴咧开到巨大，打了个响鼻。
<br/>　　“来啊，老子现在出来了，有种你就放开她冲我来啊。”
<br/>　　穷奇突然飞起来，悬停在半空中，然后……做出了一件十分残忍的事情——他嘴巴一张，用力合了起来。
<br/>　　“不要！”
<br/>　　当叶少阳意识到它要做什么、拼命冲上去时，穷奇的嘴巴也合上上，两排犬牙，瞬间穿透了秋莹的身体，青绿色的好像植物汁液一样的液体，从数不清的伤口中一下子涌了出来，秋莹微张着嘴巴，努力把脑袋转过来，朝叶少阳看过来，然后又朝大家望去。
<br/>　　她的眼神，充满了留恋和不甘，但是没有绝望，她最后看了叶少阳一眼，那表情，似乎想要微笑，但嘴角刚牵动，眼睛就闭上了。
<br/>　　穷奇舌头一卷，将她吞入腹中，嘴巴再张开的时候，只有一堆精魄从他嘴里飞出来。
<br/>　　秋莹死了。
<br/>　　几秒钟之后，听到小白等人的哭声，叶少阳才认识到了这个事实，就这么……突然地就死了。
<br/>　　“野兽永远是野兽……”赤月罗刹暗中跺脚，抱怨了一声。
<br/>　　明明有个人质在手中，只要控制好了，不愁叶少阳他们妥协，结果穷奇非要把人杀了，激怒了叶少阳，新一轮的战斗不免又要开始了。
<br/>　　“啊——”
<br/>　　叶少阳爆发出一声愤怒的吼叫，拔出七星龙泉剑，朝穷奇冲了过去，小九现出真身，飞奔起来，高高跃起。
<br/>　　叶少阳跳到她身上，小九背着他，扑向了悬在半空中的穷奇。
<br/>　　穷奇刚吞了修为不错的秋莹，心中的杀意被勾了起来，扇动翅膀，朝叶少阳扑过去。
<br/>　　“杀了他们，为秋莹报仇！！”
<br/>　　捉鬼联盟的成员们一个个脸上流着泪，冲出结界，奔着太阴山大军而去。
<br/>　　他们人很少，在黑压压的大军面前，根本不值一提，但他们的脚步没有丝毫地迟疑，在秋莹死的那一刻，对他们而言，这就成了一场不需要结果的战斗，战斗的意义只有一条：为同伴报仇。
<br/>　　至死方休。
<br/>　　因为穷奇被叶少阳对上了，其余人就直奔赤月罗刹等人去了，反正他们都是主谋或帮凶，没一个是无辜的。
<br/>　　十几个人，就像十几颗重磅诈弹，在大军中炸开了。
<br/>　　轰……
<br/>　　一道血光，在空中炸开。
<br/>　　叶少阳紧紧抱着小九的脑袋，被震飞了一段距离，止住了势头，看那穷奇，也被他们震飞出去，随即也扇了几下翅膀才在空中站稳，朝他们看了过来，眼神中露出诧异。
<br/>　　他实在没想到，刚才这一个回合的交锋，自己竟然没占到什么便宜，这一人一妖，实力之强，远远超过了他的预料。
<br/>　　不过，这个结果并没有让他感到害怕，反而激起了他好斗嗜杀之心，在空中调整了一下之后，他再度俯冲下来。
<br/>　　小九轻轻落地，也再度跳起，背着叶少阳，与穷奇在空中大战起来。
<br/>　　穷奇固然厉害，但叶少阳是上仙牌位，加上小九这个妖王，双方一时间也不分胜负。
<br/>　　双方越打越到了海上，又一次硬碰硬的攻防下来，小九泄了一口气，往下方落去，一低头才发现自己在海上，还是打的太激烈了，注意力完全在这上面，别的竟然忽略了。
<br/>　　快要落水的时候，水中突然翻起一道巨浪，一个巨大的东西从冲出水面，是那只水蛭——血池罗刹的真身。
