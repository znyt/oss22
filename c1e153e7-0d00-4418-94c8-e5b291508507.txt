<br/>　　而对面的蒋念则是做出了四个多兰戒和一瓶血药，在装备上面要领先杨小艺一个戒指。
<br/>　　而杨小艺这边不够出多兰戒，所以只能选择草鞋这件装备来给自己提供一些移速了，她也没想着要留钱攒到下一波。
<br/>　　因为这个时候任谁都知道，这一波上线等到双方到达六级的时候，基本上不是你死就是我亡的局势了，双方肯定要拼一波，尤其是杨小艺这一边。
<br/>　　她现在补刀已经落后对方了，如果在继续这样下去的话，等到对方率先补到一百刀，那么还是自己输。
<br/>　　片刻后，场上两人身上几乎同时白光一闪，瞬间两人全部到达了六级。

<br/>　　在w技能位移上去之后，杨小艺轻巧的一个扭身直接躲过了蒋念的q技能。
<br/>　　事实上这一波蒋念对于杨小艺的攻击方式，也是吃了一惊，手上的动作慢了一拍，所以才会出现这样的情况。
<br/>　　在蒋念的眼里，他认为杨小意到达六级的时候肯定会利用自己的qw来打自己的，或者是直接qr，他从来没想过对方会这样白白浪费一个w技能。
<br/>　　w技能不用来打伤害，就是想要靠近自己一段距离，不过在失去先手的情况下，蒋念这边目光倒是更加的凝重了起来。
<br/>　　看着自己q技能的冷却时间，蒋念在心中飞速的计算着自己的伤害，计算着自己再少了一个q技能的情况下，到底能不能够击杀对方。
<br/>　　就在他思考着的时候，视线中已经是看到妖姬出手了，q技能的印记已经打在了他的身上，紧接着大招瞬间开启，在这之后妖姬的链子已经是打了出来。
<br/>　　看着这一幕，蒋念第一时间交出了自己的闪现，虽然现在自己比妖姬多了一个多兰戒，但是妖姬这个英雄的爆发他还是承认的，对方恐怕就算是在少一个多兰戒的情况下，也是能够秒杀自己的。

<br/>　　然而就在蒋念闪现的一刹那，他突然发现妖姬刚刚的那个e技能居然不是朝着自己原来的那个位置释放的，而是朝着妖姬右侧一点的位置释放的。
