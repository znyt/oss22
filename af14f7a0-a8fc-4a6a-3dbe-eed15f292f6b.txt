　　萧逸已离开洞府尽头，回到主局之内。
<br/>　　洞府尽头的那古朴巨大房间，方才是古帝的真正坐化之处，长眠之地。
<br/>　　不过，那里留下的三份奖励，已经被悉数拿走了。
<br/>　　一份是无数年前洛前辈拿走的破晓钟。
<br/>　　一份是一个神秘女子拿走的重宝。
<br/>　　在那位炎殿前辈的灵识口中，那是个小丫头，至于是谁，没有详说了，拿了什么，也不得而知。
<br/>　　不过，很显然，那个所谓的小丫头，也必是个老妖怪级别的人物。
<br/>　　而拿走的东西，必然也非凡物。
<br/>　　第三份，也就是最后一份，被称作最无用的奖励，那数十万滴绝世妖兽精血，则被萧逸取了。
<br/>　　三份奖励，已被取走。
<br/>　　另外，18道古帝禁制，也已被萧逸悉数破除。
<br/>　　属于古帝洞府的真正机缘，古帝的武道圣丹，则已被萧逸取得并吞服，由此武魂成功第二次觉醒。
<br/>　　也就是说，现今的洞府尽头，事实上已形同虚设。
<br/>　　洞府尽头，以后只会是再普通不过，简单不过的房间。
<br/>　　当然，那里是古帝的长眠之地。
<br/>　　不过，古帝洞府，不代表着就此消散于天地。
<br/>　　古帝洞府内，仍旧是禁制重重，非寻常人能进来染指。
<br/>　　简单来说，古帝洞府，除了真正机缘被夺外，其它的一切，都不会有变化。
<br/>　　他日，还是会继续出世。
<br/>　　主局这里，现今18道禁制巨门虽然已尽数被破；但，他日再次出世时，18道禁制巨门会再次凝聚出现，成为下一批来此的天骄的考验。
<br/>　　除却洞府尽头外，其余的地方，主局也罢，主局之外的那些禁制通道等等也罢，不会有任何改变。
<br/>　　日后，还是会有年轻一辈的顶尖天骄，前来试图破除那号称千万载岁月从无人能破的主局。
<br/>　　主局，以及整个洞府，仍旧是一块试金石。
<br/>　　而同时，这里的奖励，同样仍旧不菲。
<br/>　　中品圣器、上品圣器、绝世丹方、上古功法等等，这里应有尽有。
<br/>　　只不过，少了洞府的真正机缘罢了。
<br/>　　说起来，这古帝洞府内，仍旧宝贝无数。
<br/>　　单单那无数的珍稀天材地宝，强大功法、圣器，就足以让萧逸也为之眼馋，不，甚至是狂热。
<br/>　　若是能得到这些的所有宝贝，他直接一飞冲天，傲视中域都有可能。
<br/>　　可是，不知为何，洞府尽头的古帝禁制虽已消散。
<br/>　　但其它地方的重重禁制，却显然仍旧存在，仍旧那般古老，仍旧那般强大莫测。
<br/>　　故在重重禁制之内的奖励，萧逸即便眼馋，也根本得不到。
<br/>　　按理说，洞府尽头才是真正的古帝主禁制，一旦被破，整个洞府的所有辅助禁制，都会失效才对。
<br/>　　可现实是，这些禁制，仍旧好好的。
<br/>　　萧逸也想不通原因。
<br/>　　或许，只如他刚才思索的那般，日后的古帝洞府，仍旧会是中域顶尖天骄的试金石。
<br/>　　终究会吸引无数年轻天骄妖孽趋之若鹜来此。
<br/>　　萧逸摇了摇头。
<br/>　　古帝洞府这里的考验，无论哪一项，都显得太过苛刻了。
<br/>　　对，苛刻。
<br/>　　这一代的年轻天骄，被誉为最妖孽的一代。
<br/>　　可，所有人联手下，外加各个带队者的超强实力带领下，方才有半数人能达到主局。
<br/>　　若是换了往常的各代天骄妖孽，真正能到达主局的，不过寥寥数人。
<br/>　　而普通天骄妖孽，甚至连禁制通道都过不了。
<br/>　　而主局前，更是直接便将手段尽出的各个带队者悉数拦下，甚至有性命之虞。
<br/>　　连莫悠，这位名震中域的忘忧剑，再加得了剑帝本源的增幅，以及其余五大学宫的辅助，都仍旧奈何不了主局。
<br/>　　而且，那时候还剩下整整3道禁制巨门未破。
<br/>　　也就是说，即便年轻一辈中执牛耳者的莫悠，也还要在现在的程度上，再出色数个层次，外加其他天骄倾力相助，方才有可能真正破主局。
<br/>　　年轻天骄妖孽，每差一个层次，可就是天差地别般的差距了，更别说差数个层次。
<br/>　　可想而知主局考验，苛刻到何等地步。
<br/>　　而真正过了主局，去到洞府尽头。
<br/>　　即便是萧逸这个外人口中当世无一的双生子，甚至是等同两个身份重合下，手段尽出，也还要加上各种重宝的帮助，费了九牛二虎之力，再加拼命一搏，才堪堪过了考验。
<br/>　　古帝洞府的各处考验，当真是苛刻得变态。
<br/>　　回想起自己一路闯过考验的经历，萧逸也不禁暗暗庆幸。
<br/>　　这些经历中，特别是走到主局之后，哪怕他有一次差错，整个古帝之墓一行，都会付诸东流，彻底失败。
<br/>　　如他的雾妖，若他没有雾妖，他根本破不了妖阵禁制。
<br/>　　如老者灵识给他的幻道手段，若无，他将破不了幻道禁制。
<br/>　　还有他的那些一身重宝，随便没有一样，他都无法真正破去18道禁制。
<br/>　　18道禁制，只要一道未破，其余17道破了，也等同失败。
<br/>　　“呼。”萧逸重重地呼出一口气。
<br/>　　万幸，古帝洞府的真正考验，他已经过了，真正机缘，他也已经得到了。
<br/>　　而且，上次剑帝碑一行的事，早已给萧逸敲响了一记警钟。
<br/>　　再怎么难得的不世机缘，若他保护不了，即便得了，也只是为他人做了嫁衣。
<br/>　　故这一次，他特地吞了古帝圣丹，武魂第二次觉醒成功才离开，省得被人惦记。
<br/>　　萧逸缓缓走着，不多时，已出了主局范围。
<br/>　　手中禁制瞬间触发，身影一闪，已然彻底离开了古帝洞府。
<br/>　　这也代表着，他的古帝之墓一行，已然结束。
<br/>　　……
<br/>　　外头。
<br/>　　各方势力，并未离去，仍旧在等待着。
<br/>　　特别是傲东楼、五大学宫长老、十八府长老等人，脸色，不知何时起，已然变得惊骇到极点。
<br/>　　“越来越弱了。”傲东楼莫名其妙地吐出几个字。
<br/>　　十八府长老点了点头，“对，我们也感知到了。”
<br/>　　“周遭禁制力量，变得越来越弱了。”
<br/>　　“这在以往，是从未发生过的事。”
<br/>　　五大学宫长老，眉头紧皱，“以往古帝洞府每次出世到消失，都不曾有这种异象。”
<br/>　　傲东楼沉声道，“唯一的可能，只可能是属于古帝洞府的真正机缘，终于被人得到了。”
<br/>　　十八府长老，点了点头。
<br/>　　不必傲东楼多说，他们也明白个中缘由。
<br/>　　“该死，真被那个小贼得到了？”五大学宫长老，脸色霎时难看。
<br/>　　“不愧是双生子之一。”傲东楼，会心一笑。
<br/>　　“出来了。”傲东楼看向禁制前方，笑了笑。
<br/>　　前方，一道冷傲的身影，从禁制中凭空而现，随后缓缓走出。
<br/>　　正是萧逸。
