　　直接赶往中州需要半年的时间，走空间虫洞去中州也将近需要二十天左右。
<br/>　　在天涯城中游玩过，也见过空间虫洞的构造，新鲜感散去，整个行程都变得枯燥起来。
<br/>　　好在有慕清霄在，根本不需要担心安全问题，小医仙与彩鳞才能够安心的修炼，对于周围的事情充耳不闻。
<br/>　　空间虫洞中的速度虽比徒步赶路快的多，但枯燥也更加浓郁，映入眼中的都是千篇一律的黑暗。
<br/>　　慕清霄仰躺在船只的甲板上微微闭目，同时操控着船只，小医仙与彩鳞则一左一右盘坐在两旁修炼。
<br/>　　在慕清霄的控制下，船只速度也越来越快，随后就化作一道银色流光划过空间，朝着深邃的黑暗通道中疾射而去。
<br/>　　……
<br/>　　……
<br/>　　中州，一座葱郁的辽阔平原上。
<br/>　　平原的中央位置，有一片碎石广场，广场中心位置的地面上绘制着诸多玄奥符文，符文上银光流转，隐隐透着空间之力。
<br/>　　广场中寂静无声，伴随着一阵寒风呼啸而过，广场中心位置顿时浮现一个银色光圈，一条船只自虚空中穿出，旋即三道身影出现在地面上。
<br/>　　“这里就是中州吗？”
<br/>　　站在广场中央，小医仙感受着周围寒冷的天气，好奇的问道。
<br/>　　彩鳞也是默默的打量着周围的环境，道：“如果空间虫洞连接的地方是中州的话，估计是没错了。”
<br/>　　“这里就是中州，看来传送阵地点并非城内，难道你们没有察觉到，空气中灵气的浓郁程度，远远不是天涯城所能比拟的。”
<br/>　　说着，慕清霄目光在碎石广场上扫视一圈，道：“这里的空间虫洞也远比天涯城的稳固，我们先找个落脚地在说吧。”
<br/>　　话语刚落，慕清霄神识扩散而出，不多时，空间一阵扭曲，三人的身影就已经消失不见。
<br/>　　几十里外，有一片无限延伸的城墙，城墙通体雪白，用特殊石料建造而成，如此巨大的城墙工程难以想象。
<br/>　　城墙内是一座繁华无比的城市，城市占地极广，从城门口放眼望去，一眼望不到尽头。
<br/>　　城墙内一条小巷中，空间轻轻波动了一下，旋即三道身影凭空而现。
<br/>　　“空间挪移真是一如既往的好用。”
<br/>　　慕清霄的目光在街道上闪过，繁华的街道上人来人往，斗王斗皇级别多不胜数，甚至有许多身穿炼丹师长袍的身影。
<br/>　　中州果然不愧是中州，加玛帝国千里范围内，想要见到炼丹师，估计要去主城才能见到。
<br/>　　但是，在这里却随地可见。
<br/>　　“哇，这座城市好大啊。”
<br/>　　小医仙望着背后望不到边的城墙，捂嘴惊呼道。
<br/>　　彩鳞眼中也是闪过一抹精芒，在视线中，强者居然随地可见，就在几里外她就能感受到一股与自己相差不远的气息。
<br/>　　“走吧，我们先找个酒楼。”
<br/>　　“本王需要转化能量，在过几日就能完成，你们逛吧。”说着，彩鳞直接化作七彩吞天蟒的姿态，裹在他的手臂上。
<br/>　　慕清霄有些无奈，只好拉起小医仙的玉手，顺着人流汇入街道上。
<br/>　　走在繁华的街道上，听闻周围行人的谈论声，慕清霄已经知道自己等人的所在地，这里是中州中域。
<br/>　　中域指的自然是中等区域，而中域中让人每天脍炙人口谈论的事情，无非就是关于冰河谷，焚炎谷与极其神秘的音谷。
<br/>　　中域酒楼林立，顺着人流很快就看到一家生意不错的酒楼。
<br/>　　踏进酒楼，一名身穿灰袍的仆役就走上前来，面露微笑道：“两位客人，里面请。”
<br/>　　瞥了一眼灰袍仆役，一个大斗师在这里居然只能当一个仆役，这便是偏僻乡村与大城市的区别。
<br/>　　找了靠窗口的位置，慕清霄看了一眼桌上的菜谱，点了两壶美酒与一些招牌菜肴。
<br/>　　菜谱上的菜肴有的居然是用高阶魔兽肉做的，斗灵级别魔兽的肉不少见，斗王级别也不是没有，难怪酒楼生意如此的受欢迎。
<br/>　　“你们知不知道，冰河谷正在招收特殊体质的弟子，待遇非常的好。”
<br/>　　“最近音谷那边似乎出现了高阶灵药，疑似八阶灵药，有斗宗强者前往，结果有去无回，音谷一如既往的神秘啊。”
<br/>　　“我还听说，焚炎谷谷主一直在找高阶炼丹师，似乎要七品以上，不知道究竟所为何事？”
<br/>　　听着大厅中的客人们谈论纷纷，慕清霄嘴角牵起一抹笑意，其中倒是有一些有用的信息。
<br/>　　冰河谷似乎与魂殿有牵连，但只要不招惹他，慕清霄也无心去理会一个小小的冰河谷。
<br/>　　至于音谷向来神秘，似乎不参与争斗，但暗地里的摩擦估计不会少。
<br/>　　最后就是焚炎谷了，正巧身在中域，没有不去焚炎谷的道理，要知道焚炎谷中可是有他需要的异火‘九龙天罡火’。
<br/>　　焚炎谷，中州‘一殿一塔二宗三谷四方阁’中的三谷之一，其谷祖先焚炎老祖曾经达到九星斗圣巅峰的实力，后因不明原因陨落。
<br/>　　焚炎谷现任谷主为唐震，实力为五星斗尊，太上长老火云老祖实力为二星斗圣。
<br/>　　若说三谷孰强孰弱，拥有斗圣坐镇，焚炎谷绝对排在前面，暂且不说音谷，冰河谷只有冰尊者，似乎并没有斗圣，只能排在最后。
<br/>　　不多时，酒肉饭菜上齐，色香味俱全，并且有一股浓郁的能量蕴含其中，不由的引来客厅中无数人侧目。
<br/>　　能够在酒楼中吃五阶魔兽肉，身份肯定是不会低，顿时引来一道道羡慕的目光。
<br/>　　慕清霄给两只酒杯中倒满好酒，道：“仙儿，酒水与五阶魔兽肉中都含有丰富的能量，多吃点能助你修行。”
<br/>　　“嗯。”
<br/>　　对于慕清霄的话，小医仙乖巧的点了点头，拿起酒杯抿了一口，随后便感觉有一股能量钻入体内，将她绝美的脸颊染成绯红一片，煞是可爱。
<br/>　　感觉口腔中有些辣味，小医仙有些不明白，男人为何都喜欢喝酒。
<br/>　　不过，感受到体内的能量，她即刻将一杯酒水灌入口中，随后运转起心法，开始默默的炼化。
<br/>　　见她的模样，慕清霄嘴角抽了抽，
<br/>　　……
