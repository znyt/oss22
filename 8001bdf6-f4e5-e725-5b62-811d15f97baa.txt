　　一个外门弟子，神力境武者，敢说要杀了夜天君。
<br/>　　只怕是白萱要是从别人的口中，听到这个消息，都会笑岔气来。
<br/>　　夜天君是谁。
<br/>　　北邙学院弟子一代的第一人，核心弟子，先天神体，乃是无数先天神体中的佼佼者，混乱神体！
<br/>　　这混乱神体，便是在无数先天神体之中，都位列前十的恐怖体制。
<br/>　　楚风眠的实力更是达到了生死境巅峰的恐怖人物，论起真正实力，甚至都隐约要超越生死境。
<br/>　　别说是一个外门弟子，就算是内门弟子，种子弟子，敢说去对付夜天君，简直都是白日做梦。
<br/>　　但是现在，从楚风眠的话中，却是透露出了无边自信，这自信，让白萱都有些一愣，似乎无法反驳。
<br/>　　“此人的资质，的确奇高，也许成长之后，并不会弱于夜天君，他未来就算是无法抗衡夜天君，也会在北邙学院中有着一番成就。”
<br/>　　“如今选择跟他交好，倒是不错。”
<br/>　　白萱心中暗自想到。
<br/>　　楚风眠的性格，她倒是已经了解了一些，是非分明更可贵的是，楚风眠的性格，不会像是许多武者一样，为了利益，而出卖朋友。
<br/>　　楚风眠向来看重情谊，这倒是许多武者中最为可贵的一点，跟楚风眠交好，起码不会被楚风眠暗地里捅刀子。
<br/>　　所以白萱才会放心来拉拢楚风眠，邀请楚风眠一起探索虚空海，寻常那颜古龙帝的洞府。
<br/>　　“寒月漓的事，虽然轰动，不过现在北邙学院中，却是有着另外一件更加轰动的事，不知道你有没有听说。”
<br/>　　白萱故意卖了个关子，开口道。
<br/>　　楚风眠心中自然清楚，这自然是万雷帝劫的事，不过楚风眠的脸上却是装出了一副茫然的样子，随意开口道。
<br/>　　“哦？我刚从圣山离开，不知道北邙学院中又出什么事了？”
<br/>　　“看来你还不知道，不过你在圣山中修炼，倒是正常。”
<br/>　　白萱缓缓的开口道，她的眼神中，也满是震惊。
<br/>　　“不是我们北邙学院的事，是整个北域，甚至是九域都发生了一件大事，便是有着一尊天才，度过了万雷帝劫。”
<br/>　　“万雷帝劫？是我们北邙学院的弟子？”
<br/>　　楚风眠脸上故意流露出了几分震惊道。
<br/>　　“是谁？”
<br/>　　“不知道，并非是我们北邙学院的弟子，如今北邙学院的无数长老，也已经纷纷离开学院，探察了。”
<br/>　　白萱道。
<br/>　　“甚至是许多学院中的老古董，都纷纷出世了，去寻找这到底是谁度过了万雷帝劫。”
<br/>　　“此人应该并非是来自那一大势力，所以不止是我们北邙学院，地魔门，北域中的无数势力，也都是在寻找着此人的踪迹。”
<br/>　　白萱的言语之中，虽然有着许多羡慕嫉妒，但是最多的却是一种叹息。
<br/>　　“此人虽然度过了万雷帝劫，但是却引来了不少灾祸，只怕是不管他加入哪一方势力，都会被源源不断的暗杀，追杀吧。”
<br/>　　“度过万雷帝劫的天才，要是成长起来，绝对足以影响现在北域的局势，没有任何一方势力，愿意诞生出如此可怕的强者出来。”
<br/>　　听完白萱的话，楚风眠的心中也是冷汗之流。
<br/>　　本来度过了万雷帝劫，楚风眠的心中还是无比喜悦的，但是现在听到白萱的话，一下子点名出来，楚风眠也才明白。
<br/>　　他虽然是从中得到了无数的好处，但是更多的，却是危险。
<br/>　　能够度过万雷帝劫的天才，一旦是成长起来，足以影响北域，甚至是影响整个九域的局势。
<br/>　　就像是楚风眠属于北邙学院，他若是将这个消息公布出去，北邙学院肯定是会大力培养楚风眠的。
<br/>　　但是，像是地魔门这等，跟北邙学院敌对的势力，知道了这个消息，可也会疯狂的去暗杀，追杀楚风眠的。
<br/>　　就算是保护的在周全，面对一方庞然大物势力的追杀，也唯有陨落一条。
<br/>　　也幸好楚风眠并没有将这个消息，透露出去，他也明白这度过万雷帝劫的轰动性。
<br/>　　只是如今听完白萱一番话，楚风眠才清楚，这个消息远远是比起他想象之中的还有可怕的多。
<br/>　　“我度过万雷帝劫的消息，绝不能够暴露。”
<br/>　　楚风眠的心中暗自想到。
<br/>　　他度过万雷帝劫，是在无尽虚空之中，所以不可能有人知晓，只要是楚风眠自己不暴露出去，便是安全无比了。
<br/>　　而且现在楚风眠还只是神力境的境界，并非是达到生死境，所以就算是怀疑，也不可能怀疑到楚风眠的身上。
<br/>　　不过这倒是给楚风眠提了个醒，现在楚风眠身上的奇遇是越来越多，但是一旦暴露出去，也会极为麻烦。
<br/>　　人皇罐，战龙决，造化之力，在加上现在楚风眠度过了万雷帝劫。
<br/>　　这任何一件事暴露出去，都会引起九域震动，楚风眠必须要保护好这些秘密才行。
<br/>　　“其实你突破的时间，跟那万雷帝劫出现的时间，极为接近。”
<br/>　　白萱开口的一刻，也是不由的多看了楚风眠几眼道。
<br/>　　“我当时还有些怀疑，那个人就是你。”
<br/>　　“只不过你只是突破神力境，不可能引来天劫，你要是那个时候突破生死境，那我可就觉得，那个人肯定会是你了。”
<br/>　　白萱这番话，虽然是在闲聊，但是其中，却是似乎隐约暗藏一些试探。
<br/>　　楚风眠心中冷笑，脸上却是露出笑容道。
<br/>　　“怎么可能，万雷帝劫？我度过生死大劫的时候，能够引来百雷王劫我就很庆幸了。”
<br/>　　“万雷帝劫可不是我这等寻常武者可以引来的。”
<br/>　　“你也不用如此客气，你的资质其实不比寒月漓低多少，将来引来千雷皇劫还是有着极大的可能的，倒时候你也会受到学院的培养的。”
<br/>　　白萱笑了笑道。
<br/>　　“算了，不谈这些了，你我现在就连神力境的极限都没有达到，更何况谈生死大劫。”
<br/>　　“走吧，在聊下去，估计他们要等急了。”
<br/>　　“我们预定约定的地点，是在虚空海的外围，我们先过去吧。”
<br/>　　“虚空海，终于要一睹真容了。”
<br/>　　楚风眠心中露出几分渴望，对着传说中的虚空海，他也早想一看了。
<br/>　　“走！”
<br/>　　两道遁光，瞬间从北邙学院中消失。
