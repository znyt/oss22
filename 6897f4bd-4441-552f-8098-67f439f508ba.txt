　　王之暴君是高阶异种，并刺杀何培源失败死在猎者联盟总部的这一消息，在猎者联盟强大的信息网覆盖下，一下子传播到了各个地方，连远在南方边境的段氏基地，都在第一时间得知了此事。
<br/>　　大多数猎者都在这一消息中震惊无比，他们为之崇拜的那个高级猎者，竟然是个异种？
<br/>　　天大的笑话！
<br/>　　紧接着，很多人又忍不住心慌起来，对不少出名的猎者产生了怀疑，那些从不露出真面目的神秘猎者，会不会也是混进人类阵营的异种？
<br/>　　在世界各地都为了这一消息沸腾的时候，猎王却皱眉：“这么简单就死了？哪怕是白允儿，也不可能吧？”
<br/>　　一旁的丁思尧不解问道：“那货死了你还愁眉苦脸的干什么？狼牙的高级情报里可是有记录的，王之暴君不仅是异种，还是异种王！”
<br/>　　猎王看着他：“就是因为是异种王，我才奇怪，能跟你们老大争锋相对了这么多年的异种王，会这么容易死？”
<br/>　　丁思尧眨了眨眼睛：“你怀疑其中有诈？”
<br/>　　猎王忽然想到了什么，道：“去查下战力总排行榜。”
<br/>　　丁思尧也双目一亮，转身就走：“我马上让情报部的人去查！”
<br/>　　——————
<br/>　　白家大宅内，白忧也在第一时间得知了这一消息，他的眼神冰冷，淡淡的吐出了两个字：“废物。”
<br/>　　“对！那个异种王就是废物，这么点小事都做不好！”白清愤怒的咒骂道：“竟然还死在了猎者联盟，那何培源也必然没杀掉吧？这种废物竟然也能称王？”
<br/>　　白忧淡然转身，冰冷的双眸直视白清：“我是说，你是废物。”
<br/>　　白清镇住，惊讶不已的张大嘴巴，不可思议间竟一句话都说不出。
<br/>　　怎么变成了她是废物？
<br/>　　白忧冷着脸，心中有股类似于火山般想要喷发的怒火，异种王当然不可能这么搞笑的死在猎者联盟的总部内，还闹的人尽皆知？
<br/>　　他让人给猎者联盟内的猎王发布了一则任务，以恐怖的价钱请他去南方边境暂守段氏基地，这本来是一个没有任何漏洞的调虎离山计。
<br/>　　可偏偏，王之暴君竟然擅自行动，在白忧的这则任务还未发布之前，就率先接下了一个莫名其妙的B级任务，让猎者联盟内的人起疑。
<br/>　　所以才会发生之后的猎王离开，猎者联盟总部依旧有援兵的情况，这时候过去刺杀何培源不是找死么？
<br/>　　这么大的漏洞暴露出来，异种王要是还能继续刺杀，那也直接说明了某个问题。
<br/>　　根本就是将计就计！
<br/>　　白忧在得知异种王这些行为之后，便已经想通了之前他没在意的某些细节，同时被戏耍后的恍然大悟，更让白忧忍无可忍。
<br/>　　区区卑劣的异种，竟然敢玩弄他？！
<br/>　　愤怒让白忧的表情带着一丝扭曲，更让他即将到达临界点的怒火，急需找到一个发泄的目标！
<br/>　　于是在他走出屋外没多久……
<br/>　　轰！
<br/>　　一声爆响，猛然炸开。
<br/>　　他身后这堂皇的大殿，就这么忽然变成了一堆废墟，漫天的碎屑在空中飞舞，某些还沾着丝丝腥红。
<br/>　　白家人都被惊动，而后慌张的向此处跑来。
<br/>　　“大少？”
<br/>　　“发生了什么？”
<br/>　　“这个气味是……”
<br/>　　有人闻到了废墟里极淡的血腥味，喧闹的声音也就此安静下来，白家人都低着头，不敢再继续问下去。
<br/>　　白清死了，哪怕她在白家身份不低，哪怕她所在的旁支仅次于嫡系，但若是白忧出的手，便没有人敢多问一句。
<br/>　　——————
<br/>　　猎者联盟总部四周环山的某处山峰上，皑皑白雪之中大·片的黑色显眼至极，那是一只只沉默安静的丧尸，等阶都在五阶之上，数量多到惊人。
<br/>　　它们安静无声的站在雪地里，很多丧尸都被大雪埋没了双膝，但它们毫无所觉，就像是死了一般的一动不动。
<br/>　　在这海量的丧尸群最前方，有几名装扮一模一样的黑袍人站在山崖边缘，不难猜测这是异种，等阶还很高，否则身后这数量恐怖的丧尸，是谁控制着它们一动不动？
<br/>　　而在这几名黑袍人之中，却还有一人有些与众不同，这是一名有些清瘦的年轻男子，他穿着很不同的衣服，与其他异种不同，他的手腕和脖子暴露在外，面上更没有任何遮挡物。
<br/>　　他的面容带着一丝清贵，还有一股难以形容的鲜活感，很奇怪更相互矛盾。
<br/>　　最重要的是他的眼睛，并不是异种惯有的猩红色，而是与人类并无差别的深褐色，只是若贴近了仔细观察，却也能看到他瞳孔深处的那抹深褐色，似乎带着一点红。
<br/>　　“我喜欢这个地方。”年轻的男子开口，咧开嘴露出两排洁白的牙齿：“我要把这里，改造为新的异种大本营。”
<br/>　　身后几名装扮无差别的异种纷纷单膝跪地，朝着这名年轻男子行礼：“听从王的号令。”
<br/>　　这名与人类毫无差别，甚至连牙齿都完全一致的年轻男子，才是真正的异种王木叶。
<br/>　　此时一名异种恭敬道：“眼下王之暴君的事情已经人尽皆知，王您又已经到达了十阶，完全蜕变成了人类的样子，可以更加肆无忌惮的伸展手脚了吧？”
<br/>　　木叶偏头看了这名异种一眼，眼中带着一丝玩味：“蠢货才会相信王之暴君死了，你忘了战力排行榜了么？”
<br/>　　听闻这一细节，几名异种纷纷大惊：“那这次假死，岂不是白费心思？我们可是为之付出了一名九阶巅峰的异种代价啊！”
<br/>　　“谁说白费？”木叶又咧嘴一笑：“猎者联盟中的王之暴君是异种，不代表战力排行榜上的王之暴君是异种，更不代表跟白家接触的那个就是异种王。”
<br/>　　听着木叶弯弯绕绕的话，几名异种都接不上话，他们已经不记得从什么时候起，木叶就这般的真真假假，有时候连很多异种王的身边人，都认不清到底哪个才是他们的异种王。
<br/>　　“哪怕知道我是异种王又怎么样？”木叶说着，手指在自己的唇上一点，笑容诡异：“有人说，可是有人信么？难道要刨开我的身体，取走我的心脏，吃一口尝尝？”
<br/>　　“我已经和人类毫无差别！”
