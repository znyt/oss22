<br/>				    四人的眼中皆划过一抹异色，只是一瞬间，在闭关修炼的他们依旧盘膝静坐着，然，神魂却是离开身体往那悬浮之地而去。<br/>　　<br/>　　    在那悬浮之地，凤九在破开第一道禁制之后，只感觉天空中的那股迫压感消失了，当下，她再度凌空而起，准备破开这里的大阵，然而就在这时，四道威严的声音伴随着凌厉之势从天空中传来。<br/>　　<br/>　　    “你是何人！”<br/>　　<br/>　　    那声音威严而蕴含着凌厉之势，强大的威压自那四道声音中传出，震得她心神一震，一丝鲜血从口中溢出，而底下的众人则在那四道声音不约而同传出之时，纷纷口中喷出鲜血的趴跪下去，连直起身体都办不到。<br/>　　<br/>　　    这一刻，他们方知那四位仙帝的强大远非他们所能想象，也许，经过那么多年，这四位仙帝的实力已经不仅仅只是仙帝级别了……<br/>　　<br/>　　    抬手拭去嘴角溢出的鲜血，凤九看向那从天空之中探出来的四抹身影，眸光微闪，声音不亢不卑的从口中传出：“凤九，见过四位帝君。”<br/>　　<br/>　　    四人的睿智而凌厉摄人的目光落在那一袭红衣的凤九身上，哪怕对方是以男装打扮，但以他们的修为还是一眼便看出了对方是女儿身，而且还是一名玄灵之体的鬼才。<br/>　　<br/>　　    目光在她的身上掠过，见她容颜绝美，一身气质尊贵不可亵渎，身上并无黑暗气息，更无煞气环绕，相反的，此人隐隐身上有圣光出现，让四人心下微异。<br/>　　<br/>　　    “凤九，你因何会进了这流放之地？又为何要破我四君联手布下结界大阵？”<br/>　　<br/>　　    其中一人询问着，声音威严而摄人，目光紧紧的锁定着凤九，直视着她的清眸，仿佛要将她看个通透一般。<br/>　　<br/>　　    这地方是流放之地，所进来的都是恶人居多，就算偶尔有一两个被传送进这里的，也不应该会是像她这样的人。<br/>　　<br/>　　    更何况，这地方是他们四帝君联手布下的，以她小小一女娃，居然能破得开他们这里的禁制，当真是让他们大吃一惊，觉得不可思议，不由的，猜测起她的来历。<br/>　　<br/>　　    “我是被一魔修打开的黑暗之门传送进来的，到了这里已经快一年了，破四君的结界大阵，自然是想要离开这里。”她缓声说着，一面看着那四人的影子。<br/>　　<br/>　　    这是神魂出窍了，得怎样的修为才能做到这一点？果然如她猜测，这四人早已不在仙帝级别了。<br/>　　<br/>　　    听到她是因黑暗之门而进来的，四人目光微闪，原来是如此。没想到魔修之中竟还有人有能力打开黑暗之地传送阵。<br/>　　<br/>　　    微顿了一下，其中一人便开口道：“此处乃是关押恶人之地，若是破开这里的结界，势必他们会重回大陆为恶，既然你是误入此地，我等可送你离开。”<br/>　　<br/>　　    听到这话，底下众人心头一惊，他们趴跪在地上无法站起身来，更无法抬头看向天空，因而此时，心中不免生出一股绝望。<br/>　　<br/>　　    凤主是会答应吧？反正她能离开，他们能不能离开，于她来说也不是很重要的事……<br/>
