　　而在官方谈判进行的如火如荼这段时间，情报科也没闲着，大量的顶级报情员被派了出去，潜伏于各大商人的工厂和家里，时刻注意着他们的动向，而在长安城里更是到处都布满了情报科的眼线。
<br/>　　西市一间不起眼的小店里，黑子与称心相对而坐，在他们的面前摆着两盏早已经变冷的茶水。
<br/>　　良久之后，黑子才缓缓开口：“红衣，现在咱们情报科的人手严重不足，你真的就不能回来帮帮忙？”
<br/>　　“陛下已经放了我自由，我为什么还要回来？”称心淡淡的说道。
<br/>　　现在的他已经不是当初那个被李元昌到处送人的傻孩子，倭国十余年死在他手下的倭人没有一千也有八百，如今的称心早就已经厌倦了这样的生活，只想着找一个没人认识自己的地方老老实实的度过余生。
<br/>　　黑子沉默了一会儿，他当然清楚称心的想法，可是犹豫良久之后还是继续说道：“红衣，我知道倭国的经历让你十分厌倦这个职业，可你毕竟咱们情报科出去的人，如今咱们情报科人手不够，你难道就不能帮帮忙？”
<br/>　　“你想让我怎么帮，这么多年，我除了杀人就是杀人，分析情报并不是我善长的，这样的事情你应该去找杨天！”称心的语气很平静，但却斩钉截铁的拒绝了黑子。
<br/>　　“杨天我当然要找。”黑子双眼紧紧盯着称心：“但是你的作用没有人可以替代，情报科现在需要你。”
<br/>　　“不，你把事情想的过于复杂了。”称心推桌而起，走到小店的门口，看着外面的夜色淡淡说道：“陛下曾经说过，这个世界其实少了谁都无所谓，第二天太阳依旧会照样升起，人们该知饭还是要吃饭，该工作还是要工作，没有人是不可或缺的！”
<br/>　　“你……”黑子看着称心的背影，突然不知应该说些什么，当年的孩子如今已经开始步入中年，已经不是他能左右的了。
<br/>　　“去找杨天吧，他会帮你们的，至于我……，你就当我不存在好了。”称心说到最后，迈步走出小店。
<br/>　　“等等！”眼看称心就要离开，黑子也站了起来，厉声说道：“红衣，你真觉得自己可以就这样脱离情报科么？你不要忘了当初的誓言，你的一切都是属于大唐的！”
<br/>　　称心并没有回头，只默默看着脚下的石板路，幽幽说道：“是的，我的一切都属于大唐，但是科长，我并不属于情报科，也不属于你！”
<br/>　　黑子这下是真没招了，眼看着称心就要离开，他知道这次如果不将他搞定，下一次就更没有机会，于是追出两步说道：“我从来就没有说过你属于我，也没说过你属于情报科，但是眼下各路奸细云集长安，你难道就这么眼睁睁的看着他们在你的眼皮子底下折腾？”
<br/>　　“我累了，想退出！”
<br/>　　“好，那你退出吧，你走，你走好了，我就不信没了张屠夫还能吃了带毛的猪，你走吧！我黑子从今天开始，就不认识称心这个人！”黑子这次是真的死心了，撂下一句狠话之后头也不回的转身就走，不消片刻人已经消失在夜色之中。
<br/>　　称心低着头，呆呆的站在无人的街道上，听着身后小店打烊，伙计安装窗板的样音，深深吸了一口气，艰难的迈出一步。
<br/>　　……
<br/>　　长安城，大安坊，一个紧靠着城墙的偏僻角落里有一座不大的宅子。
<br/>　　宅子很小，小到只有一间房，两个房间，院子里放不下一辆马车的程度。
<br/>　　深夜中，整个大安坊似乎都已经陷入沉睡，可这个不起眼的小院里却依旧亮着灯，一个小小的影子透过灯光映在窗纸上，显的是那么孤独。
<br/>　　称心站在院子的门口，就那么静静的看着投射在窗上的影子，良久才迈开步子走进小院。
<br/>　　而就在称心刚刚走进院子的刹那，窗上的影子突然动了一下，一个稚嫩的声音传了出来：“师父，是您回来了么？”
<br/>　　“是我！”称心应了一声，穿过院子，推开了房间的门。
<br/>　　房间里面的布置十分简单，进门便是厨房，一个灶台，一口锅，几只碗，再加上两个不大的缸就是厨房中的一切，而在厨房的两侧，则有两个只挂着两个帘子的门。
<br/>　　“师父，为什么您的脸色那么难看？是不是谁惹您生气了？”随着称心进入房间，其中一个帘子被人掀了起来，一个大概只有六、七岁的女童从里面跑了出来，仰起头看着他有些阴郁的脸，关心的问道。
<br/>　　“没有！”称心摇摇头，在女童的头上轻轻按了按，强自笑了笑：“小钰，今天的功课做完了没有？”
<br/>　　“早就已经做完了！”女童十分乖巧的点点头。
<br/>　　“既然做完了就早点休息吧，明日还要去学堂呢。”
<br/>　　“嗯！”女童小钰答应了一声，往回走了两步又有些担心的回头看了称心一眼：“师父，您真的没事么？弟子看您的脸色好难看。”
<br/>　　“师父没事，小钰去休息吧！”称心摆了摆手，并没有对小钰解释刚刚发生了什么，直到女童转身进了自己的房间，才默默的叹了口气，转身进入另外的一间屋子。
<br/>　　一张木榻，一张小几，一床被子，便是称心房间中的一切。
<br/>　　当年名动倭国，让人闻之色变的‘红衣’称心，竟然住在如此简陋的地方，如果让人知道了估计一定会惊掉下巴。
<br/>　　但是称心却没有任何一点的不适，就那么合衣躺到榻上，将手枕在自己的脑后，微微闭上了双眼。
<br/>　　晚上发生的一切历历在目，黑子的每一句话都让称心受尽煎熬。
<br/>　　他称心并不是没有心，也不是不知道感恩，如果他现在依旧是孤身一人，很可能他头也不回的就跟着黑子离开。
<br/>　　可是他现在有人需要守护，那个可怜的丫头小钰就是他的一切牵挂，他不能丢下这个小家伙离开，所以他只能拒绝黑子，哪怕是被黑子误会也无所谓！
