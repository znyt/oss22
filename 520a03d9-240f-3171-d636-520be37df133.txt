　　倭人被杀的落花流水，消息传回长安之后，举国欢庆！
<br/>　　大唐现在已经不怎么打仗了，但兵锋所指依旧所向无敌，大唐的百姓渴望胜利，大唐的军人渴望胜利，大唐的皇帝陛下同样渴望胜利。
<br/>　　拜占庭等国使节闻之大唐大胜的消息并不意外，不过出于礼节，还是向李承乾前交了贺书，言词间虽不谄媚，但也表现出了恭敬之意。
<br/>　　李承乾对此并不怎么在意，命人备了回礼之后便将此事丢在了一边。
<br/>　　时间已经是四月，那些收到邀请国家的使者也都纷纷来到长安，一些正式的商业洽谈也即将展开，那些繁文缛节的事情就暂时先不要计较了。
<br/>　　……
<br/>　　长乐、杨雨馨还是小武这段时间一直都很忙，不，应该说忙的不可开交。
<br/>　　那些拜占庭、大食等国的使者毕竟不是自己来的，跟着他们一起的还有很多异族的商人，这些商人在长安走过一圈之后，最想要做的就是见一见大唐的‘主事之人’。
<br/>　　异族番邦虽然也是贵族为先，但是却没有士、农、工、商这样的等级排位，异族商人在本国的地位与大唐商人在大唐的地位是截然不同的。
<br/>　　他们的商人很多本身就是贵族，所以在他们的国家商人见国王也不是什么难事。
<br/>　　大唐虽然有气节，不会按照异族番邦的规矩做事，但第一次见面的情况下，皇室多少还是要给这些异族一点面子的。
<br/>　　所以就在大唐迎宾楼，长乐和小武摆下了酒宴，请那些番邦贵族商人吃饭，当然，这顿饭其实也并不是那么好吃的，来的人也不只是番邦贵族，也有大唐的商人。
<br/>　　……
<br/>　　迎宾楼现在是长孙皇太后的产业，李承乾虽然当了皇帝，但就算他有天大的胆子也不敢把他老娘的产业改到自己名下。
<br/>　　再说迎宾楼的那一点点收入，连李泰现在的都有些看不上，长乐更是理都不理，至于李承乾……好吧，这一位现在富有四海，需要什么东西都是用抢的，他老娘的那个迎宾楼赚那点钱都不够他手指缝里露的多，真的没有必要去惹老太太不高兴。
<br/>　　不过，虽然这几位皇族看不上这些产业，但迎宾楼的奢华依旧让那些异族番邦看的目瞪口呆。
<br/>　　那些被他们在家里精心呵护的瓷器在大唐竟然只是用来装骨头的盘子；那些被他们用来炫耀身价的丝帛在这里竟然只是被当成抹布来用；那价值千金的玻璃制品，在这里竟然只是普普通通被人随手摆在一边。
<br/>　　一个拜占庭商人曾亲眼看到一个侍女失手打碎了十来个晶莹剔透的玻璃杯子，在他看来这足以让那个侍女丢掉性命。
<br/>　　可是，最终的结果却是，那个侍女随手招来了一个伙计，将那些碎玻璃收拾了出去，就好像什么事情都没有发生过一样，甚至那个收拾玻璃的伙计都没有表现出任何的异样来。
<br/>　　天啊！这到底是一个什么样的世界？大唐到底要富裕成什么程度！要知道，那些精美的玻璃杯子如果放到拜占庭，仅一只就可以换取数十个最强壮的奴隶，如果是十来个杯子……，苍天啊，五百个强壮奴隶就这么被打死了！
<br/>　　拜占庭商人虽然知道玻璃制品就是由大唐生产出来的，在本地一定会十分便宜，但他还是禁不住产生一种想要杀人的冲动。
<br/>　　哦对了，还有头顶的那个所谓的电灯，那东西简直就是，简直就是神的恩赐，璀璨、耀眼，却没有任何奇怪的味道。
<br/>　　这需要多少钱？到底需要多少钱？大唐的皇太后难道是大唐最富有的人么？否则的话怎么可能会有如此多奢华的东西布置在这里。
<br/>　　疯了，真是疯了，在等待着大唐长公主殿下以及皇妃殿下的这段时间，所有异族商人几乎都快要被眼前的一切吓傻了。
<br/>　　这些人不是没有见过世面，而正是因为见过世面，所以才会对眼前的一切产生一种强烈的不可置信。
<br/>　　桌上的美食是他们见所未见过的，侍女身上的礼服依旧是他们从未见过的，电灯这种东西依旧是他们从来没有见过的。
<br/>　　那些异族的贵妇们尽管已经身状盛装，但是与那些身着礼服，待人接物彬彬有礼的侍女们比起来，似乎她们才更像是侍女。
<br/>　　无数的贵妇嫉妒的看着那些侍女，那闪着光的眼睛如果能将其中的光芒转化为实质的话，估计早就已经把那些侍女身上的礼服抢走了。
<br/>　　大唐有句成语怎么说来着？哦对，叫不虚此行！这一次大唐来的还真是不虚此行啊，见到了太多以前想都没有想过的东西。
<br/>　　来时那种可以一连半月都不停歇的车子，路上那种一个人就可以骑着走却怎么也不会倒下的车子，这些无一不在让那些异域贵族感到自惭形秽。
<br/>　　而与之相对的，大唐商人则自然了许多，他们很多都是迎宾楼的常客，已经习惯了这里的一切。
<br/>　　那些在异族看来十分珍稀的东西，在他们看来还真不算什么，玻璃杯子的价格很便宜，三、五文钱就可以买上好几个，至于说那些瓷器，好吧，其实不少商人家里就是干这个的，那些瓷器他们自己就可以烧制，不过就是一些土而已。
<br/>　　不过这些事情他们是不会与那些异族人说的，这是商业机密，来此之前他们就已经商量好了，绝不把这些秘密透露给那些外族知道，否则就是大家的公敌。
<br/>　　……
<br/>　　宴会还没有开始，但是随着外面天色逐渐变的暗淡，所有人都知道，两位大唐的顶级贵族马上就要到了。
<br/>　　一个是大唐长公主殿下，一个是大唐皇帝的妃子，这两个女人几乎掌握着小半个大唐的经济，绝对是在场所有人中最高不可攀的存在。
<br/>　　只是不知道这两位殿下何时会出场，又会不会答应接下来众人提出来的贸易请求，如果她们不答应的话那真是太可惜了，而如果她们能够答应……好吧，那就发大财了！
