　　“你们败了。”
<br/>　　萧逸淡淡地走到三人面前。
<br/>　　天残地缺、黑无常三人无力地躺在地面，再无战力，连站起来的力气都没有。
<br/>　　使用了升龙，加上灵器的增幅，萧逸的实力本就达到了半步洞玄，甚至稍强一些。
<br/>　　也就说，在真气这个角度，他完全不输这三人。
<br/>　　再加上绿色武魂释放的火焰，二十道火刃融合的爆炸攻击，其威力之强....
<br/>　　连萧逸有着修罗战体第一层圆满赋予的强悍肉身，加之立刻逃离，仅仅是承受了爆炸的余威，尚且吐血，受了不轻的伤势。
<br/>　　更何况天残地缺三人，肉体本就比不上萧逸，又是近距离承受爆炸，不被炸成重伤才怪。
<br/>　　十道火刃的融合，已经不亚于噬火百刃大成。
<br/>　　二十道火刃，显然同样不是一加一等于二那么简单，威力绝对是十倍提升。
<br/>　　若是三十道火刃融合，必然又是十倍强于二十道火刃融合。
<br/>　　难怪当初在狂血玄君的洞府时，五十道火刃融合，竟炸得那些禁制屏障都激荡起来。
<br/>　　萧逸甚至能想象到，若是有朝一日，自己能将一百道火刃完全融合，噬火百刃绝对会超越玄阶的范畴，起码达到地阶武技以上的层次。
<br/>　　地阶武技，恐怕整个北山郡都没有几本。
<br/>　　这时，天残地缺三人听着萧逸淡淡的话语，脸色一阵变幻。
<br/>　　他们竟然败了？暗影楼四大杀手，竟然败得如此彻底。
<br/>　　以往就算是天残地缺、黑白无常分别出任务时，也不曾这么狼狈过。
<br/>　　如今，四人联手，竟还是败了，而且是败于一个少年手中。
<br/>　　半晌，他们的脸色恢复正常。
<br/>　　“败了？败了便败了。”黑无常说道。
<br/>　　“我们技不如人，杀不了你，让你离开便是。”天残地缺同时说道。
<br/>　　“呵。”萧逸淡淡一笑，“一句放我离开就想了结此事？”
<br/>　　“不然呢？”黑无常冷眼看着萧逸。
<br/>　　“你还打算留下我们不成？”天残地缺满是不屑之色。
<br/>　　“当我不敢杀你们？”萧逸反问一声。
<br/>　　“你敢？”黑无常冷声道，“若我们今日死了，你便等着暗影楼的高手亲自来追杀你吧。”
<br/>　　天残地缺露出了傲然之色，“我们暗影楼乃是北山郡最强势力之一，你以为就只有我们区区四个半步洞玄吗？”
<br/>　　“哼，楼中多的是洞玄境，乃至强悍的破玄境。”
<br/>　　“若今日我们三人死了，北山郡，将再无你易霄的容身之地。”
<br/>　　“你易霄声明崛起不过数月，天才之名刚刚显露，难道要变成丧家之犬，承受我们暗影楼无穷的追杀吗？”
<br/>　　三人威胁起来，脸色中带着冷意和不屑，丝毫不惧萧逸会杀了他们。
<br/>　　天残地缺、黑白无常，确实是暗影楼四大金牌杀手，但只局限于百武城附近一带。
<br/>　　暗影楼作为北山郡的大势力之一，杀手遍布北山郡，自然有更强的武者。
<br/>　　“小子，最好不要做傻事，识相地便放了我们。”三人挣扎着从地面上站起来，边说着，边准备离开。
<br/>　　“哼。”萧逸冷哼一声，脚步一动，一掌打向黑无常。
<br/>　　“噗。”黑无常猛地吐出一口腥血，瞬间被击飞，当跌落时，已经生机全无。
<br/>　　萧逸震碎了他的心脉。
<br/>　　“你...”天残地缺顿时一惊，惊骇地看着萧逸，“你当真不怕我们暗影楼报复？”
<br/>　　“笑话。”萧逸冷笑一声，“若真放你们走了，你们暗影楼的杀手才会来找我麻烦。”
<br/>　　“你什么意思？”天残地缺面露怒色。
<br/>　　“你们的鬼话，骗骗别人还行，骗我？差远了。”萧逸冷声道，“暗影楼确实有洞玄境乃至破玄境的杀手，可那些层次的强者，绝非一群山贼首领能请动的。”
<br/>　　“他们出的价钱，顶多能请动你们四个。若任务失败，赏金自动退还；若你们在任务过程中死了，暗影楼便会没收赏金，作为你们四人死掉的赔偿。”
<br/>　　“你们死了，那些山贼亏大了吧，能否再凑够钱请动洞玄境以上的杀手还不一定呢。”
<br/>　　萧逸满脸戏谑之色地看着天残地缺。
<br/>　　“你胡说。”天残地缺嘴硬道，“我们若死了，楼中强者绝对不会放过你。”
<br/>　　“还在骗我。”萧逸摇了摇头，道，“暗影楼可是个杀手组织，若没有利益，你们楼中的杀手肯出手？呵呵，你当我是傻瓜，还是当自己是傻瓜。”
<br/>　　“杀手这个职业，若不是为了利益，谁会去当。若没有利益，谁会出手。当然，也有特殊情况。或许，暗影楼楼主是你爹，或许，你在暗影楼中有背景。”
<br/>　　萧逸嗤笑一声，手中凝聚出了一股火焰。
<br/>　　后面的两句，他只是随便说说。若这四人真有背景，也不至于混得那么差。
<br/>　　天残地缺瞪大了眼睛，满脸不可置信，“易霄，你怎么会对我们暗影楼如此清楚，难道你曾经是我们楼中的杀手？”
<br/>　　“不...不可能，你小小年纪，绝不会是杀手。”
<br/>　　两人看着萧逸手中的火焰，面露惊恐之色，“不...你不能杀我们...”
<br/>　　他们的话没能说完，萧逸的火焰已经脱手而出，将两人击杀。
<br/>　　至此，在百武城附近一带凶名赫赫的暗影楼四大杀手，天残地缺、黑白无常，死。
<br/>　　“呼...”萧逸吐出一口浊气，微微压下了体内的伤势。
<br/>　　这场大战，原本他可以更轻松地结束。只不过，他更想酣畅淋漓的战斗一番。
<br/>　　至于伤势，不重也不轻，影响不大，过些天就会自动痊愈。
<br/>　　这时，他微微转过身，准备离开。
<br/>　　恰在此时，地面忽然一阵剧烈的颤动，震动了起来。
<br/>　　“嗯？”萧逸眉头一皱，脚步移动，迅速后退。
<br/>　　下一秒，两条粗壮的藤蔓蓦然从地底飞速升出。
<br/>　　藤蔓极巨大，缠绕起来，几乎将整个道路完全封锁。
<br/>　　“阁下杀了我暗影楼的人，就想如此离去不成？”
<br/>　　一道飘渺的声音，忽然在空气中飘荡。
<br/>　　萧逸嘴角咧过一道冷笑，“阁下在暗处看了那么久，连天残地缺被杀都不出手，我还以为阁下会一直不现身。”
<br/>　　萧逸早就发现，还有一人藏在暗处。只不过此人实力似乎很强，他也是在差不多结束战斗时才发现的。
<br/>　　他早就知道还会有一场大战，所以并没有管天残地缺四人的尸体，也没有拿他们的乾坤袋。
<br/>　　他佯作离开，只是逼这暗中之人出现。
<br/>　　“那四个废物，只会给我们暗影楼丢脸罢了，我自然不会出手救他们。”
<br/>　　空气中再次传来声音。
<br/>　　“想要易某的性命，便现身吧。”萧逸淡淡地说道。
<br/>　　半空中，忽然无数绿叶缓缓飘落。
<br/>　　绿叶飘絮，如雪花飘舞，甚是美丽。
<br/>　　只不过，这美丽之中，却隐藏着致命杀机。
<br/>　　凡是被绿叶接触到的事物，尽皆被切割而开。
<br/>　　嘶嘶嘶，旁边一块巨石，不出数秒便被切割成一堆碎石。
<br/>　　“嗯？”萧逸眉头一皱，身上火焰澎湃而出，在自己身边的绿叶快速烧尽。
<br/>　　“倒是有些本事。”
<br/>　　这时，一道身影，在这无数绿叶中飘然而下。绿叶围绕着他，如同一只只绿色精灵。
<br/>　　观其样貌，竟是个翩翩公子。
<br/>　　年龄不大，约莫17、8岁。面容俊朗，气度不凡。
<br/>　　“你是？”萧逸面露疑惑。
<br/>　　“钟无忧。”
<br/>　　“钟无忧？”萧逸皱起了眉头，半晌，微微吃惊，“钟无忧？暗影楼少楼主？”
<br/>　　萧逸不禁暗暗无语，刚才自己还在嗤笑暗影楼楼主是天残地缺四人的爹，没想到，现在人家儿子还真来了。
