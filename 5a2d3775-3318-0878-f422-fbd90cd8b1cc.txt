　　“一定是假的！这绝对不是真的度厄金丹！”
<br/>　　柳天生在心中狂吼着，
<br/>　　然而，
<br/>　　随着他将神念，注入丹药之中，脸色顿时越来越难看，
<br/>　　最终，“噗通！”一声，
<br/>　　柳天生一屁股坐倒在椅子上，满脸的失魂落魄，口中喃喃道：
<br/>　　“不可能！这怎么可能！丹药品质竟然如此精纯，即便是五品炼丹师，能够做到如此程度的，也是凤毛麟角啊！”
<br/>　　“难道四公子的背后，竟然有一位五品炼丹师，甚至更高？”
<br/>　　听到柳天生这话，一旁的张齐云同样坐不住了，
<br/>　　“五品炼丹师？那种大人物，怎么可能会出现在这小小的天玄星域？不！这不可能！”
<br/>　　张齐云脸色惨变，
<br/>　　倘若张凌云的背后，真的有一位五品炼丹师的支持，那他还怎么比？
<br/>　　“查！立即给我去查！务必要弄清楚，老四的这些四品上等丹药，到底是出自何人之手？”
<br/>　　……
<br/>　　与此同时，
<br/>　　张凌云这边，却早已经乐开了花，
<br/>　　“恭喜四公子！贺喜四公子！光是今天一天，咱们这边的拍卖场，就有一千万块灵石的进账，如今我们一天的业绩，已经将二公子远远甩在了后面！”
<br/>　　“是呀！有这位叶大师在，四公子的少阁主之位，算是稳了！”
<br/>　　“要说这位叶大师，还真是厉害，比二公子府里的那位柳大师，简直强大太多了！”
<br/>　　……
<br/>　　听到众人的吹捧，张凌云顿时越发得意，
<br/>　　“嘿嘿，这位叶先生的手段，的确出乎我的意料之外，我倒要看看，这一次老二还怎么和我争？只要我赢了这次考验，就算父亲和各位长老，也无话可说！”
<br/>　　众人纷纷称是，唯独那位雷叔，沉思不语，似乎并不怎么乐观。
<br/>　　张凌云似乎也看出了这一点，将其他人支走之后，方才道：
<br/>　　“雷叔，你是不是有什么话要对我说？”
<br/>　　雷叔点了点头道：
<br/>　　“四公子，并非老夫要泼你的冷水，诸位长老之所以要设下这个考验，显然是想让你知难而退，如今你虽然占据了优势，只怕有些人要坐不住了！”
<br/>　　张凌云眉头一皱，
<br/>　　“雷叔，你说的是寿天禄那个老东西？”
<br/>　　雷叔脸色微微一变，连忙道：
<br/>　　“四公子慎言！寿长老乃是我万宝阁的太上长老，更是化神天君级的大人物，虽然老阁主更喜欢你，可是却也不得不顾及此人的面子！当年老阁主能登上阁主之外，便是多亏了此人啊！”
<br/>　　张凌云顿时一脸恨恨道：
<br/>　　“是呀！所以这老东……他就千方百计的打压我，好让他的外孙成为下任阁主！哼哼！”
<br/>　　原来，张凌云和张齐云并非一母所生，张齐云的外公，便是万宝阁的太上长老寿天禄，同时也是张齐云的最大靠山。
<br/>　　雷叔幽幽叹了一口气道：
<br/>　　“为今之计，我们也只能希望，寿长老不要插手小辈之间的争斗才好！”
<br/>　　……
<br/>　　万宝阁张府，最深处，一个犹如世外仙境的院落之中，陡然传出一道苍老的咆哮之声，
<br/>　　“废物！烂泥扶不上墙！枉我费尽了口舌，舍下了这张老脸，方才说服长老团，设下了这次考验，没想到你竟然如此没用，真是把我的老脸都给丢尽了！！”
<br/>　　这咆哮之人，是一身紫色长袍的白发老者，赫然正是万宝阁的太上长老寿天禄。
<br/>　　而那个被骂之人，不是旁人，正是二公子张齐云。
<br/>　　张齐云等到寿天禄骂的差不多，方才一脸委屈，道：
<br/>　　“外公，这事真的不能怪我啊！我也没有想到，那老四手下，竟然有人的炼丹术，比柳大师还要强！极有可能是一位五品炼丹师！”
<br/>　　“五品炼丹师？”
<br/>　　寿天禄听到这话，顿时眉头一皱，
<br/>　　“我天玄星域，什么时候出了五品炼丹师？我怎么从来没有听说过？”
<br/>　　张齐云连忙将自己打听到的情况，一五一十的讲了出来。
<br/>　　寿天禄听完，眉头顿时皱的更紧了，一脸的难以置信之色，
<br/>　　“你的意思是说，那小子的修为只有金丹境，却能轻松击败贺长老？而且还能炼制十种不同的四品上等灵丹？你糊弄鬼呢！”
<br/>　　张齐云顿时哭丧着脸，
<br/>　　“外公，你这是什么话，我糊弄谁，也不敢糊弄您老人家啊！此事很多人亲眼目睹，的的确确是真的啊！”
<br/>　　寿天禄眉头紧皱，不再多说，直接一道神念释放出去，瞬间便将诺大的万宝阁，全部笼罩其中。
<br/>　　很快，
<br/>　　寿天禄锁定了一道陌生而神秘的气息，想要仔细探查一番，
<br/>　　轰！
<br/>　　神念竟然被反弹开来！
<br/>　　“嗯？”
<br/>　　寿天禄两眼猛然一蹬，自言自语道：
<br/>　　“好啊！没想到老夫的眼皮子地下，竟然藏着这么一个高手，我竟然丝毫没有察觉！”
<br/>　　“只不过，以此人的实力和手段，为何会插手我万宝阁的家事，难道他想通过扶植老四，来达到控制万宝阁的目的吗？”
<br/>　　张齐云听到这话，顿时不惊反喜，连忙道：
<br/>　　“外公英明！此人来历如此神秘，如今躲在我们万宝阁，肯定是有所图谋，您老人家可不能不管啊！干脆直接出手杀了他……”
<br/>　　“哼！”
<br/>　　还没等张齐云没有说完，寿天禄顿时冷哼一声，将他下面的话直接打断，道：
<br/>　　“胡闹！此人虽然来历不明，可是却没有做任何对不起万宝阁的事情，而且又是老四的请来的客卿，我身为万宝阁的太上长老，若是贸然对他出手，让别人怎么看我？你不要脸，我还要呢！”
<br/>　　张齐云顿时一阵颓然，
<br/>　　“那这该如何是好？难道就任由老四这么嚣张下去，那我们之前的努力，岂不是都白费了？若是让老四成了万宝阁的继承人，我还不如一头撞死得了！”
<br/>　　寿天禄两眼一瞪，
<br/>　　“你急什么？我又没说不管！”
<br/>　　说到这里，寿天禄微微一顿，接着又道：
<br/>　　“算你小子运气好！过两日，丹域的百草仙人，将会驾临天玄城，我就再舍下这张老脸，请他老人家出手帮你一次就是了！”
<br/>　　张齐云听到这话，顿时满脸狂喜之色，
<br/>　　“外公说的可是那位，在丹域有药王之称的百草仙人吗？据说他老家人，在五品炼丹师中，也是最顶级的存在啊！”
