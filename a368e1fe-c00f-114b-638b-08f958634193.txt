        童筱颖端坐在床上，双手捂在腹部下方，只觉两腿间刚刚被傅夜沉碰过的地方还有一丁点的撕痛感。
<br/>
<br/>        刚刚他那是几个意思？
<br/>
<br/>        就那样完事了吗？
<br/>
<br/>        童筱颖忽然间有些猜不透傅夜沉的心思了。
<br/>
<br/>        算了，不猜了。
<br/>
<br/>        她好心按婆婆的意思，想要跟他上床，结果他还不领情。
<br/>
<br/>        亏她还对他小鹿乱撞来着……
<br/>
<br/>        童筱颖从床上起来，换了身衣服，洗漱完后，便被女佣直接叫去跟冷夫人和傅老太爷一起用午膳。
<br/>
<br/>        吃饭的时候，饭桌上极其安静，直到管家进来通报说家庭医生来了，傅老太爷才放下碗筷，看向童筱颖和冷夫人说道：“我吃好了，你们俩慢慢用。”
<br/>
<br/>        “爸，如果身体有哪里不舒服，一定要告诉我。”冷夫人有些担忧地看向傅老太爷。
<br/>
<br/>        童筱颖也随之看过去，只见傅老太爷傅炎廷微微点了点头。
<br/>
<br/>        这位爷爷的岁数应该有八十多岁了吧！
<br/>
<br/>        但身子骨看上去还挺硬朗的。
<br/>
<br/>        傅老太爷离开后，冷夫人才将目光落在童筱颖的身上，关心地问：“昨晚，小沉可还喜欢？”
<br/>
<br/>        童筱颖听着冷夫人这问话，差点噎到，一旁候着的女佣见状，连忙给她端上了茶水。
<br/>
<br/>        嘴中的食物和着茶水一起咽下肚后，童筱颖放下手中的茶杯，对冷夫人赧然一笑：“他、他很喜欢。”
<br/>
<br/>        喜欢才怪！
<br/>
<br/>        看到她穿成那个样子，他不但没像视频里那样跟她上床，反而对她变得若即若离起来。
<br/>
<br/>        童筱颖不知道这是不是自己的错觉，总觉得，昨晚之后，傅夜沉起初回来时，对她的言行举止都很奇怪。
<br/>
<br/>        “你们俩既然经历了这么多事情，就应该彼此好好珍惜彼此。”冷夫人话中有话地说，顿了顿后，又接着说道，“筱颖啊！这段时间，总是要你陪着我做这做那的，你都没时间去做你自己喜欢的事情。刚好这几天，我没什么事情需要你陪着，你就去做你喜欢的事情吧！”
<br/>
<br/>        “哦！”童筱颖先是讷讷地应了声，随即又微笑着补充了两个字，“好的。”
<br/>
<br/>        冷夫人会意地对童筱颖笑了笑。
<br/>
<br/>        午膳用完后，冷夫人会去午休。
<br/>
<br/>        童筱颖独自回房去待了一会儿，觉得很是无聊，于是随手挑了一个手提包，塞好了手机和现金，还有傅夜沉给她的卡，离开了傅家大庄园。
<br/>
<br/>        她自己徒步出的门，没有叫司机开车送她，出门前倒是跟管家林贵叔打了声招呼。
<br/>
<br/>        沿着傅家大庄园外面的主路出发，在不知不觉中，童筱颖走到了市中心的步行街。
<br/>
<br/>        大概因为今天是周末的缘故，街上的人比往常要多。
<br/>
<br/>        童筱颖走走停停，遇到自己喜欢的小玩意或者是小吃，就会毫不犹豫地买下。
<br/>
<br/>        现在，无论她去哪里，做了什么，一切动静，哪怕傅夜沉不在国内，他都知道得一清二楚。
<br/>
<br/>        与此同时，连华生正在自己的厂子里和团队做研究。
<br/>
<br/>        而他研究的东西，就是傅夜沉交给他的“解药”，换句话说，就是他父亲的研究成果，他现在在和自己的手下拿来做成品分析。
<br/>
<br/>        实验刚进行到一半，连华生接到了夏玄灵打来的电话，说她想跟他单独见面，是有关于裴金澈的事情。
<br/>
<br/>        连华生不得不终止实验，脱掉了白马褂，去赴夏玄灵的约。
<br/>
<br/>        夏玄灵约连华生在滨江城里一家地处偏僻的咖啡馆里见面，很明显，她不想让熟人碰巧遇见她的行踪。
<br/>
<br/>        连华生见夏玄灵这神神秘秘地，进了咖啡馆，在夏玄灵对面坐下后，忍俊不禁地打趣道：“我说夏家少小姐，你是被人追杀了吗？搞得这么神秘！”
<br/>
<br/>        “华生，除了你，我真不知道该找谁说。”夏玄灵顿时六神无主起来。
<br/>
<br/>        连华生顿时收敛了脸上那漫不经心的笑容，转而严肃地看着夏玄灵，问道：“发生什么事了吗？”
<br/>
<br/>        “大概一个月前的样子，死而复生的童筱颖来找了阿澈。还给了阿澈一盒什么药，还跟阿澈说了一些奇怪的话。阿澈自己给自己注射了那药，结果……”夏玄灵柳眉微蹙，欲言又止。
<br/>
<br/>        “结果怎么了？”连华生也跟着皱起了眉头。
<br/>
<br/>        夏玄灵忍不住抬起手来比划道：“阿澈确实恢复了感觉神经，并且彻底失去了自愈能力，知道‘痛’是什么滋味，但同时，阿澈他、他……”
<br/>
<br/>        “你能一口气把话说完吗？”连华生听着都感到有些替她着急。
<br/>
<br/>        夏玄灵一脸无奈地看向连华生：“阿澈他整个人都变了。”
<br/>
<br/>        “整个人变了？！”连华生诧异地瞪圆了眼睛反问，“什么意思？”
