　　没错，他是在笑，虽然没有笑出声来，但喜形于色，九霄琉璃焱跳动，伤口以肉眼可见的速度恢复。
<br/>　　慕清霄笑容转眼消失不见，神色变得凝重无比，虽然伤口依旧在恢复，但只有他知道，速度明显比刚才慢了一些。
<br/>　　凤凰虽然能浴火重生，不死不灭，但真的遭到毁灭性的攻击，伤势严重到无法恢复，依旧会死！
<br/>　　“呼......”
<br/>　　伤势逐渐恢复，慕清霄强忍着痛苦口中突出一口浊气，捏了捏拳头，兴奋道：“真是恐怖的力量，只差一步，只差一步就是化神期了！”
<br/>　　轻轻在空中一踏，慕清霄已经来到百米高空上，伤势已经恢复，胸膛上的肌肉洁白如玉，宛若新生婴儿，虽然白嫩，可力量感却十足。
<br/>　　抬起头来，星辰一般的眼眸中，更加恐怖的黑色雷霆，自天际倾泻而下！
<br/>　　不知何时，慕清霄的瞳孔中已经浮现黑色原子图案，永恒的万花筒写轮眼，开启写轮眼后，终于能够捕捉到雷霆的轨迹，再也不是无迹可寻。
<br/>　　望着那越来越近的黑色雷霆，慕清霄双拳紧攥，羽翼扇动，瞬间化作剑芒，猛的向雷霆迎了上去。
<br/>　　在两次雷霆的攻击下，慕清霄的肉身变得更加强横，即使体内没有阴阳灵气，他的攻击力也绝对远超元婴期修士。
<br/>　　没有花哨的招式，也没有恐怖的灵气波动，只凭肉身，宛如携带风雷，一拳狠狠的轰在迎面而来的暗黑色雷霆上。
<br/>　　“砰！”
<br/>　　两者相撞，可怕的余波，顿时以慕清霄为中心，铺天盖地的席卷而出，下方森林中的树木，瞬间被强大的气压，压成一堆碎屑。
<br/>　　四周狂风四起，风沙漫天！
<br/>　　此刻，站在上空，慕清霄喘着粗气，眼中尽是疯狂，右臂由先前的白嫩变成焦黑一片，黑色雷弧布满全身......
<br/>　　上空，肆虐的能量风暴在持续了几分钟后，渐渐的消散于天地间，而脚下的山林早已满目疮痍，黑云散去，温暖的阳光自天际倾洒而下。
<br/>　　“噗噗...!”
<br/>　　雷劫结束，慕清霄紧绷的身心也放松下来，最终还是忍不住喷出一口鲜血，意识开始模糊，黑色雷霆的最后一击，其中蕴含的恐怖能量超出了他的预料。
<br/>　　心中苦笑一声，此时筋疲力尽的他，连手指都难以动弹，也不知道恢复需要多久，或者能不能恢复。
<br/>　　渐渐的意识沉睡，慕清霄的身体也自高空坠下，犹如坠落人间的仙神！
<br/>　　自几百米高空坠下，要是全盛时期的他，连毛都不会掉一根，但现在的他早已遍体鳞伤，外表虽然看不出来，但五脏六腑皆是移位。
<br/>　　若真是这样坠落在地，绝对是伤上加伤！
<br/>　　然而，就在他落地的瞬间，异变突生，九霄琉璃焱瞬间蔓延全身，将他三百六十度包裹了起来。
<br/>　　恐怖的黑色雷弧，在九霄琉璃焱面前，也瞬间被覆盖了过去。
<br/>　　在九霄琉璃焱的燃烧下，周围空间都扭曲起来，下一秒慕清霄额头的菱形晶体闪烁几下，待降落在地面的时候，出现的不是一具身躯，而是一颗蛋！
<br/>　　没错，确实是一颗蛋，只是这颗蛋却是与众不同，蛋高达两丈，整体呈现黑色，是一种极其深邃的黑，比刚才的黑雷劫更加深邃。
<br/>　　蛋壳表面上是一条条金色的纹路，散发着神圣的气息。
<br/>　　......
<br/>　　......
<br/>　　随着时间的推移，转眼就过去半个月的时间。
<br/>　　漆黑的空间中，慕清霄意识逐渐恢复，睁开朦胧的双眼，光线暗淡，片刻后，终于确定自己似乎处于一个黑暗且狭小的空间中。
<br/>　　脑海追忆起来，回想起半个月的黑雷劫，慕清霄道：“我记得当时，在黑雷劫一击下，自己重伤垂死，现在看来，我的命连上天都夺不走啊！”
<br/>　　回归神来，慕清霄有些惊诧，自己的双眼哪怕在黑暗中，也犹如白昼，但在这片狭小的空间中，却是漆黑一片，什么都看不到。
<br/>　　连忙查探了下自己的身体情况，片刻后，慕清霄心中大喜，身体已经完全恢复，而且经脉比先前足足宽了两倍！
<br/>　　最主要的是，他现在的修为，已经正式迈入化神期，化神一重初期，而且气息很稳定！
<br/>　　深吸了口气，慕清霄压下激动的心情，目光向四周看了看，依旧是漆黑一片，根本不知道伸在何处，得想办法出去才行啊。
<br/>　　空间比较狭窄，伸手就能触摸到周围温热的障碍，慕清霄一愣，难道是石壁将自己压在地面了？
<br/>　　想到此处，猛的一拳就砸在墙壁上，伴随着‘咔嚓’一声清脆的声响，慕清霄一愣，自己可是化神期，随手一拳就能轰炸一座小山坡。
<br/>　　可这墙是怎么回事，怎么可能这么硬！
<br/>　　脑海中也没有仔细想那么多，只是想着尽快离开此处，也不知道自己恢复用了多久，于是，慕清霄又是一拳轰了上去，力量是刚才的两倍。
<br/>　　“咔嚓！”
<br/>　　坚硬的墙壁碎裂开来，明亮的光线终于照射进空间中，慕清霄总算是松了口气，看来自己并没有被困住。
<br/>　　想来也是，若是化神期的修士被困死在石壁中，传出去恐怕会笑死诸天万界的修士吧！
<br/>　　手臂直接一挥，恐怖的力量瞬间将周围的壁障击碎，刺眼的光线全部倾洒进来，慕清霄抬起手掌看了一下，怔了一下，脑子瞬间有些不够用。
<br/>　　映入眼中的不是白皙的手指，而是一只纯黑色的爪子，没错，就是一只爪子！
<br/>　　震惊过后，慕清霄缓缓起身，瞥了眼封印自己的空间，这哪里是什么石头，而是一颗刻有诡异金色纹路的黑蛋，而自己就是刚破开这颗蛋爬了出来。
<br/>　　回过神来，慕清霄连忙查看自己的身体，这一看使得他惊叫一声，但响起的却是一声嘹亮的凤鸣。
<br/>　　鸿头、麟臀、蛇颈、鱼尾、龟躯、龙纹，燕颔，与神话传说中的有些相似。
<br/>　　但比起神话传说中，慕清霄此刻的身躯全身纯黑，没有一丝杂色，唯有狭长的眼眸呈现金色，焦点则是小型的黑色原子图案，正是永恒的万花筒写轮眼。
<br/>　　整体看上去，显得神骏异常！
<br/>　　听到凤鸣过后，慕清霄久久无法回神，自己真变成了一只黑色的凤凰！
