　　“啊哈哈哈……”
<br/>　　无数诡异的笑声，此起彼伏地响彻起来，众人循声望去，却是那些阴生蔷薇，本来长的就像是人脸，现在竟然真的好像变成了人脸，一张张狰狞的脸，扭曲着，连同花枝也摇晃起来，发出可怕的笑声。
<br/>　　那些彼岸花的花瓣之中，也渗出了血液，顺着花坛，汩汩地流淌下来，蔓延到了众人的脚下。
<br/>　　“什么情况！”
<br/>　　众人大惊。
<br/>　　叶少阳对着那些血液撒出来一包朱砂，结果与黑血接触之后，发出呲的一声，将污血灼烤蒸发，暂时挡住了势头。
<br/>　　众人一时间都有点懵逼，这变故……实在来得太快了。
<br/>　　之前，叶少阳想到这株花有意引他们过来，必然有一定的把握能办了他们，不过心底还是跟大伙一样，有点不以为意：毕竟就是一棵植物，再难对付又能怎么样，现在看来，真的是想错了。
<br/>　　“先下桥，退出去再想办法！”
<br/>　　叶少阳喊了一嗓子，带头往回跑，结果一道横生的藤蔓挡住了去路，朝左右看去，桥口不知道什么时候已经被藤蔓占领：都是从那株巨花的枝干里伸出来的，顺着石桥的两边绕过来，但是没有直接进攻他们，估计也是知道一时间收拾不了这么多人，所以先挡住他们的退路，再迂回前进。
<br/>　　“我勒个去，第一次遇到这种事！”
<br/>　　看着桥面上不断伸展过来的这些藤蔓和根系，有一种好莱坞大片里的特效的感觉，不过叶少阳此刻可没工夫欣赏，深吸一口气，摸出了几张灵符，画成了地火符，然后丢了过去。
<br/>　　任何植物，不管怎么样变化多端，总是避免不了五行相克的规律，植物毕竟是怕火的，更不要说这些藤蔓都是妖孽而生，地火符一碰到上面，立刻燃烧起了大火，不过后面那些藤蔓立刻压了上来，层层叠叠，生长的速度竟然盖过了火势蔓延的速度。
<br/>　　这不科学！
<br/>　　叶少阳怔怔地看着这一幕，感觉自己遇到了前所未有的挑战。
<br/>　　“你们快来帮忙，对了梧桐，你的五宝金莲，快！”
<br/>　　叶少阳一边喊一边回过头去，霎时间，他彻底呆住了，身后，只剩下了妙心一个人，呆呆地看着自己，再没有别人。
<br/>　　叶少阳左右看去，空空如也，桥就这么宽，前面不远就是花圃，这些人……去了什么地方？
<br/>　　“人呢！”叶少阳冲妙心大叫。
<br/>　　“我……不知道。”妙心怔怔发呆。
<br/>　　“梧桐，小方！”叶少阳扯着嗓子叫起来，但是没有一点回音。
<br/>　　“他们，会不会……”妙心一脸担忧。
<br/>　　“不可能！”叶少阳知道她想说什么，但是觉得不可能，这么多人，不可能同时被抓住，更不可能一点动静都没有。不过，眼前根本就顾不上这个了，叶少阳左右看看，现在是前面有藤蔓，后面有污血，两边都在不断地顺着桥面铺展，蚕食着两人所站立的地面……
<br/>　　“快，动手啊！”叶少阳一边说，一边在脑海中寻找着对付眼前情况的办法，他是很冷静，可以算得上临危不惧，比这大的场面他也见过，只是……没有经历过这样的困局。
<br/>　　本来他是想要跳下桥的，但是两边的栏杆也都被藤蔓占领了，并且不断盘起，形成了网一样的所在，想要跳下去逃生，简直就是自投罗网。
<br/>　　没有什么好办法了。
<br/>　　“你先帮我挡着这些藤蔓！”
<br/>　　叶少阳叫了妙心一声，权衡了一下，先冲到妙心面前，把墨斗拿出来，用红线从从左侧的栏杆下面穿过，然后拉到右边，中间串上了一枚铸母大钱，然后顺着这根红线，在地上撒下了一道朱砂，布置成一道封锁线，希望能多挡一会，然后转身看去，妙心正拿着她那个魔杖一样的东西作法，不断落下流火，阻挡着不断蔓延的藤蔓，但以这些藤蔓好像复制粘贴一样的生长速度，这种程度的攻击，也只能是暂缓而已。
<br/>　　“这样不行，我们要先冲出去！”叶少阳冲到她身边，说道：“妙心，我有办法冲出去，待会你跟着我的剑走，小心一点，绝对能闯出去！”
<br/>　　妙心一怔，道：“怎么可能！”
<br/>　　叶少阳已经抽出了七星龙泉剑，被周围的阴气所刺激，七星龙泉剑亮起了炫目的紫光。
<br/>　　妙心本来是不信任，但她也是地师传人，一眼看到这把剑，感受到它的灵力，立刻就知道，这是一把九段光的神器，惊道：“这是什么剑！”
<br/>　　“我茅山的镇山之宝，七星龙泉剑！”
<br/>　　“啊！”妙心惊叫。
<br/>　　叶少阳一把拉过她，又往后退了几步，飞快地说道：“我寻思着，我们现在在这东西的地盘上，没有办法怎么样，只能被它带节奏，想要逃生，只有先冲出去，你先出去，然后寻思怎么样攻击它，从外面应该反而办法更多，我相信你！”
<br/>　　妙心震住，说道：“那你怎么办？”
<br/>　　“我在这坚持一会，没事，你听我的，时间不多，快！”
<br/>　　“这怎么行，你连宝剑都舍弃了，更不是它对手，我挡着你自己出去。”
<br/>　　叶少阳沉吟了三秒钟，说道：“我怕你挡不住！”
<br/>　　说完，转过身，望着不断袭来的藤蔓，望着手中的七星龙泉剑，沉声道：“老朋友，看你了。”当下飞快念咒：
<br/>　　“日落沙明，天地倒开，乾坤无极，道法无边！”
<br/>　　日落沙明天倒开，茅山的开天咒，配合七星龙泉剑这把举世无双的神器，所向披靡，立刻在层层叠叠的藤蔓之间，轰出了一条路来。
<br/>　　“快去！”叶少阳用力推了一把妙心，事成定局，妙心也不想浪费机会，箭步冲了出去。
<br/>　　她刚冲过去，左右的藤蔓就再度生长，从后面合上了。一切跟叶少阳想的一样，七星龙泉剑只是扫清了一条直线上的藤蔓，两人没法并排走，前后走的话，这些藤蔓不可能给他们足够的时间，走在后面的很容易被藤蔓困住，到那个时候，一切都完了。
