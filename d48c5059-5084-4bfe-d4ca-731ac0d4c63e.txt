　　东离剑宫，闭关室外。
<br/>　　“傲宫主，此番有劳了。”萧逸拱拱手。
<br/>　　傲东楼摆摆手，客气道，“易霄小友客气了，不过是借用闭关室罢了，小事一桩。”
<br/>　　“呵呵。”傲东楼边说，边笑着。
<br/>　　“我说以易霄小友的实力，何至于来此参加比拼呢。”
<br/>　　“原来竟是三位总殿主的考核。”
<br/>　　傲东楼，似是解开了之前一直想不明的疑惑，一阵得意笑容。
<br/>　　萧逸闻言，有些无奈，但也知晓，没法反驳，干脆不理。
<br/>　　拱拱手，直接进了闭关室。
<br/>　　……
<br/>　　闭关室内。
<br/>　　萧逸打出十数个屏障，封锁了周遭。
<br/>　　上次，在猎妖殿总殿那，虽然给他敲醒了一记警钟。
<br/>　　但，普天之下，又有几人能有三大总殿主那般当世传奇的修为和实力。
<br/>　　故他并不担心。
<br/>　　不过，他向来谨慎，所以还是布下了屏障。
<br/>　　东离剑宫这里，即便是傲东楼，也没那个实力轻松破他的屏障。
<br/>　　萧逸盘膝坐下。
<br/>　　只是，面具之下的脸色，仍旧无奈中，带着愤愤。
<br/>　　原本，在他的打算中，归还三大总殿的名额，只是第一步。
<br/>　　接下来，他已然有办法彻底归还三块总令。
<br/>　　只需等他结束完这边的比拼，取了奖励，就可以开始进行。
<br/>　　只是，万万没想到，三位总殿主，却抢先一步，派了灸炎前来。
<br/>　　现在，灸炎当着中域各大势力的面，宣布了此事，他萧逸，将再难寻到办法拒绝。
<br/>　　“三位总殿主，果然好手段。”萧逸双眼眯了眯。
<br/>　　“不过，不代表我便没办法了。”
<br/>　　萧逸蓦地一笑。
<br/>　　虽觉得此事麻烦，也觉得三位总殿主有些‘狡猾’，但这般博弈，事实上也有些意思。
<br/>　　故萧逸，更多的，只是觉得无奈。
<br/>　　此事，萧逸思索了一阵，便未再多想。
<br/>　　现今，修炼才是重中之重。
<br/>　　距离古帝之墓开启，还有一月多几天的时间。
<br/>　　他需要做的，是尽力地提升修为和实力，这样，才能在古帝之墓有更大的把握。
<br/>　　当然，也仅仅是更大罢了。
<br/>　　若说绝对的把握，绝不可能。
<br/>　　古帝之墓，存在了无数年，每15年一遭，至今不知过去多少次开启了。
<br/>　　而中域，历代出现过的惊才绝艳之辈，绝不在少数。
<br/>　　各大势力的天骄、隐世势力的天骄，等等。
<br/>　　可至此为止，古帝之墓，仍旧无人能破局。
<br/>　　可想而知，古帝之墓若想破局，其难度何等之大。
<br/>　　即使是他萧逸，或者莫悠、五大学宫天骄、以及现在在东离剑宫几个拥有名额的天骄，其实，都无多少把握。
<br/>　　且把握，本就低到极点。
<br/>　　各个拥有名额者，五大学宫大半年前就开始的全力培养等等，都是在尽可能地增加把握罢了。
<br/>　　萧逸现今，同样如此。
<br/>　　嗖…手中光芒一闪，取出了刚才的乾坤戒。
<br/>　　感知了一番，萧逸点了点头。
<br/>　　里头的奖励，果然足够巨额，也让他足够满意。
<br/>　　这场汇聚中域各方势力的盛事，其奖励，确实丰厚。
<br/>　　而奖励，由东离剑宫拿出。
<br/>　　东离剑宫，为了这个直接就能得到的名额，连宫内最强的剑诀副本都拿了出来，可想而知这个名额之珍贵。
<br/>　　故里头的其它修炼资源奖励，又怎可能会差。
<br/>　　说起来，当年参加百院之争时，为了一个名额，连这些学宫们都尚且诸多争执、算计。
<br/>　　可如今，他萧逸，直接便拥有了大堆名额。
<br/>　　修罗殿、风刹殿，两个。
<br/>　　猎妖殿、药尊殿、炎殿，三个。
<br/>　　算上在这里得到的一个，合共6个。
<br/>　　可惜，修罗殿的名额，他要自己保留，而风刹殿的名额，则给了张南峰和沐宁。
<br/>　　猎妖殿等三殿的名额，也归还了。
<br/>　　不然，他把这些名额卖了的心都有。
<br/>　　倒是在东离剑宫这里得到的这个名额，虽然也无用，却卖不得。
<br/>　　说起来，这就是上古八殿和各大势力的区别了。
<br/>　　各大势力、学宫，需要争夺，才能得到名额；而且各自只有5个，合共10个。
<br/>　　而上古八殿，其地位超然，直接便各自拥有一个，合共8个。
<br/>　　而且，由于这8个名额，乃是直接拥有，所有八殿给谁都可以，并无别的限制。
<br/>　　故萧逸当时给别人也行，归还也行。
<br/>　　而各大势力的名额，却不能随便给予。
<br/>　　名额的归属，定下了便是定下了。
<br/>　　故萧逸现今这个名额，也只能浪费掉了，毕竟他之后会以自己萧逸的身份去参加古帝之墓的开启。
<br/>　　用的，也只会是修罗殿的名额。
<br/>　　“呼。”萧逸深呼吸一口气。
<br/>　　嗖…嗖…嗖…
<br/>　　炼药炉，大批天材地宝、材料，被一一祭出。
<br/>　　萧逸直接开始了炼药。
<br/>　　数个时辰后。
<br/>　　待他收回火焰，停下炼药，周遭，已是一座座丹药小山。
<br/>　　“吸。”萧逸低喝一声。
<br/>　　体内两大武魂齐开，吸收速度，快到了惊人的极致地步。
<br/>　　萧逸一边吸收，一边内视着体内小世界的变化。
<br/>　　体内，那庞大小世界之内的元力，正以肉眼可见的速度，不断飙升着。
<br/>　　原本，他的修为，乃是圣皇境巅峰，达到9100道的层次；体内小世界，元力则是溢出了一成左右。
<br/>　　如今，元力不断流溢，渐渐上升到了两成、三成…还在不断飙升。
<br/>　　半个时辰后。
<br/>　　待得所有丹药被吸收殆尽。
<br/>　　体内小世界，溢出的元力，达到5成高度。
<br/>　　“足够了。”萧逸喜悦一笑。
<br/>　　以他那庞大的小世界，能将元力储存到5成高度，这已经是一个很可怕的概念。
<br/>　　不得不说，此次东离剑宫为了这个名额，果真是下了血本。
<br/>　　这些修炼资源奖励，丰厚得惊人。
<br/>　　此时，他体内小世界元力，已填满5成。
<br/>　　之后，待他离开东离剑宫，寻个地方吸收天地武道力量，便可将武道力量，提升到9500道的水准。
<br/>　　也就是所谓的踏入武道大能范畴。
<br/>　　现在，这里毕竟是东离剑宫，如果他在闭关室时，忽然落下数百道天地武道之力，终究会引人猜疑。
<br/>　　谨慎下，萧逸暂时不吸收天地武道力量，留待之后离开再吸收不迟。
<br/>　　嗖…
<br/>　　萧逸大手一挥，收了炼药炉，烧去了周遭修炼痕迹，随后起身，出了闭关室。
<br/>　　刚出闭关室。
<br/>　　一个剑宫弟子，竟是等候在此。
<br/>　　“嗯？怎么？”萧逸疑惑问道。
<br/>　　“易霄副殿主，这么快就出关了？”剑宫弟子面露惊色。
<br/>　　“寻我有事？”萧逸直接问道。
<br/>　　“哦，是的。”剑宫弟子反应过来，点点头，道，“是水凝寒少府主拜托我们宫主，让小的等候在此。”
<br/>　　“若是易霄副殿主出关了，水凝寒少府主有请。”
<br/>　　“水姑娘找我？”萧逸皱了皱眉。
