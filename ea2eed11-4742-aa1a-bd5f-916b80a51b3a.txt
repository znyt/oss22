　　叶少阳在内心深处大喝一声，将体内罡气都运转到右手上，暂时冲开了右手臂的约束，右手总算是能动了，不过行动迟缓，想要作法结印是做不到的，而且罡气耗费太多，会减弱对那股吸力的阻挡。
<br/>　　不过，叶少阳这么做并不是为了作法：他控制右手，十分艰难地把手伸进腰带里，摸出了阴阳镜，对着前方照出去。
<br/>　　“美华，出来吧！”
<br/>　　叶少阳在神识中大声叫道。
<br/>　　没错，美华是他最后的秘密武器。
<br/>　　对于这个鬼仆，好不容易遇到了，叶少阳自然不可能跟她随便分开，这几天来，她一直都没有出现，并不是不在，而是被叶少阳收到了阴阳镜里，一来是不想让别人等人知道她的存在，私收鬼仆，就算对方没资格管，也会留下不好印象。
<br/>　　陈晓宇手下有妖仆。那是他自己的事，叶少阳不想在梧桐面前降低自己的形象，而且自己这鬼仆还是个美女，很容易让人多想。
<br/>　　二来，叶少阳也是希望能多一个秘密武器，这才让她躲在阴阳镜里，神不知鬼不觉，谁也不知道她的存在，真到了万不得已的时候，再突然把她放出来，打对方一个出其不意，或许能救自己一命。
<br/>　　留下一条后路，这一向是叶少阳一贯的作风。
<br/>　　眼下就到了万不得已的时候。
<br/>　　叶少阳大声召唤美华出来救场，结果……阴阳镜里完全没有反应。
<br/>　　怎么会这样？
<br/>　　叶少阳又喊了几声，还是没有人理。
<br/>　　一丝恐惧，爬上了叶少阳的心头。难道，美华之前悄悄走了？绝不会！
<br/>　　虽然这个美华，是自己在这个世界认识的，但毕竟也是美华，性格是一样的，叶少阳非常了解美华，自己几个女性门人之中：橙子古灵精怪，小白傲娇不够成熟，雪琪从来不鸟自己，换成这三个人，都有可能半途偷偷溜出去玩了或怎么样，但是美华绝对不会。
<br/>　　美华是一个非常老实的女鬼，吩咐她做什么，就一定会坚守，她一定没有走，但是……自己这么唤她，如果真的在阴阳镜里，她也不可能不出来……
<br/>　　感受着那股吸力越来越强，几乎要将自己的法力完全吸走，叶少阳心急如焚，但是脑海中一直琢磨着这件事。
<br/>　　没有道理，美华不可能凭空失踪，这不科学……
<br/>　　“不科学”这几个字从脑海中闪现，叶少阳猛然想起了什么，之前所遇到的：一切法器都挡不住藤蔓的生长，跟永动机似的，完全不符合邪物的规律，其次，那些小伙伴们为什么一开始就不见了，唯独剩下自己跟妙心？
<br/>　　再者，自己用七星龙泉剑施展开天咒，再加上妙心自己的实力，如果这都闯不过那些藤蔓，这株巨花的修为那得有多强？
<br/>　　但是退一步说，假如它真的有那么强的修为，根本就用不着这些手段来困住自己，直接秒杀就可以了。所以……这一切都不正常，太不科学。
<br/>　　香味……
<br/>　　叶少阳猛然间想到了这个点，脑海中灵光一现，抱着试试看的心态，用力咬下了舌尖，热血流了出来，叶少阳也浑身打了个哆嗦，然后……黑暗慢慢褪去，眼前逐渐清晰，同一时间，自己的双脚也感受到了实地的接触，没有了那股强大的吸力，叶少阳定睛看着脚下，发现自己还站在桥上，没有藤蔓，也没有那些血水，一切都是不存在的。
<br/>　　果然……一切都是幻觉。
<br/>　　叶少阳抬头看去，妙心，梧桐，卢晓清，毛小方，陈晓宇和他的妖仆猴王都在，不过所有人都是痴呆一般地站着，双眼浑浊，失魂落魄的样子。
<br/>　　看到妙心好端端的，叶少阳内心顿时涌现出了一种失而复得的狂喜……虽然她不是自己什么人，但毕竟是朋友一场，看到她没事，叶少阳也是十分激动。
<br/>　　果然，一切都是幻觉。
<br/>　　叶少阳突然发现，包括妙心在内，几个人都是面色发紫，皱着眉头，一脸的痛苦，有一种快要窒息的感觉。
<br/>　　糟糕！
<br/>　　他们一定是跟自己一样，经受着幻觉的折磨，这巨花靠着香味，让他们进入幻觉，绝不可能是跟他们闹着玩的，在幻觉中死了，人多半也就挂了。叶少阳一个箭步冲上去，先奔向最近的梧桐，一只手掐住她的人中，灌入罡气，然后另只手用力在后脑勺上拍了一把。
<br/>　　“啊……”梧桐一下子醒了过来，双眼睁开，人软软地倒在了叶少阳的怀里。
<br/>　　叶少阳看她站不住，只好一把抱起她，冲到妙心面前，故技重施，把妙心也弄醒，然后费了一番功夫，把大伙都弄醒了。当轮到陈晓宇的时候，叶少阳犹豫了一下，他不是圣母婊，别人打自己左脸，还把右脸伸给他打，况且眼前这局面也不是自己造成的。
<br/>　　不过，眼看着对方有生命危险而不救，实在有点说不过去。叶少阳犹豫了一下，叫来之前被自己叫醒的卢晓清，跟他说了，卢晓清虽然对一切都不知道怎么回事，但知道陈晓宇有危险，于是按照他说的法子，把陈晓宇和他的妖仆陆续弄醒。
<br/>　　所有人都清醒过来。
<br/>　　一开始，大伙都还沉浸在幻觉中，都很迷茫，叶少阳告诉他们真相，所有人震惊，七嘴八舌地回忆幻境里的经历，都跟叶少阳一样，经历了各种光怪陆离的画面，不过有一点一样的是，所有人都经历了非常凶险的局面！如果不是叶少阳出手相救，大概所有人都得死在幻境之中。
<br/>　　别人还没什么，法师之间，互相帮助也是理所当然，更不要说还是朋友，只有陈晓宇表情有点怪异，望着叶少阳，有些踌躇。
<br/>　　“你不要谢我，是卢晓清救的你。”叶少阳脱口而出。
<br/>　　“我就知道。”陈晓宇冷哼一声，“你怎么会有这么好心。”
<br/>　　叶少阳苦笑，如果不是自己先救了卢晓清，再让他出手，只怕陈晓宇现在已经挂了，自己虽然没有出手，也算是间接救了他，陈晓宇也不是智障，这一点肯定想得到。
