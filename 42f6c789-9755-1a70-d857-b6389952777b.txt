        “我跟你说，其实我一点都不喜欢这个林若琴。她刚来的时候，对我指手画脚。要不是阿十出面，只怕她早就欺压到我头上来了。”季雅珊义愤填膺地说。
<br/>
<br/>        童筱颖在餐桌前坐下来，微笑道：“那我们是一边，都不喜欢那个女人。”
<br/>
<br/>        “当然。”季雅珊端着自己的早餐，一边说着，一边在童筱颖的对面坐下。
<br/>
<br/>        “他们两为什么会在这里？”童筱颖双手拿着三明治，咬了一口，嘴里嚼动着，说出来的话也有些含糊不清。
<br/>
<br/>        好在季雅珊与她心有灵犀：“他们两在读大学的时候，替‘冒先生’放校园贷，结果被人发现了。他们两就联手，带了几个小混混，把那人给活活打死了。后来，警察查到他俩身上了，还对他俩下了通缉令。他们两就去求了‘冒先生’，冒先生看在他俩是生化学的高材生，会制药的份上，就将他俩收留在这里了。”
<br/>
<br/>        季雅珊的话语一落，童筱颖身心怔愣，整个人突然间就像定格了一般，一动不动，就连脸上也失去了任何表情。
<br/>
<br/>        马智杰果然是他俩杀的！
<br/>
<br/>        “筱颖，你没事吧？”季雅珊将手伸了过去，抓着童筱颖的手腕，轻轻地摇了摇。
<br/>
<br/>        童筱颖回过神来，黯然神伤地回答道：“他们两杀的那个人，是一个对我很好的异性朋友。”
<br/>
<br/>        “看样子，你们是冤家路窄了。”季雅珊忍不住感慨道，顿了顿，她又安慰道，“筱颖，你别太伤心。他们两到了这里，一样的没了自由。而且，他们要是再制不出‘冒先生’想要的药，‘冒先生’会把他俩给杀了，就当是替你那个朋友报了仇。”
<br/>
<br/>        “比起让他们被那个‘冒先生’所杀，我更希望他们接受法律的制裁，去坐牢，去赎命。”童筱颖垂下眼帘，柳眉微蹙。
<br/>
<br/>        季雅珊此时不做声了。
<br/>
<br/>        因为，她不大了解童筱颖的想法。
<br/>
<br/>        何明旭和那个林若琴横竖都是没有自由、生死大权不在自己手中的人，为什么筱颖非得期盼着他们去坐牢？
<br/>
<br/>        或许两人的思想观念到不了一块，话题没有再继续下去。
<br/>
<br/>        两个人彼此自顾自地吃完早餐后，便去伺候各自的“爷”。
<br/>
<br/>        童筱颖端着早餐，前往四爷房间的途中，在古堡的长廊的拐角遇上了正在抽烟的何明旭。
<br/>
<br/>        他倚着墙壁，右手的食指和中指夹着烟，塞到双唇间狠狠地吸了一口，又放下手来，将嘴里的烟圈吐了出来。
<br/>
<br/>        童筱颖只是瞥了他一眼，正准备视而不见地只身离去时，何明旭却突然叫住了她。
<br/>
<br/>        “筱颖，我两还真的很有缘。”何明旭扯了扯嘴角，下巴上的青色胡渣，已经向她展现出他这三年来的沧桑。
<br/>
<br/>        明明他和她是同龄人，而他此刻的样子，看起来却好像比她大了十岁。
<br/>
<br/>        “在你眼里，钱重要，还是命重要？”童筱颖停下了脚步，侧身看向何明旭。
<br/>
<br/>        何明旭不假思索地回答：“钱！”顿了顿后，他又替自己辩解道，“我穷怕了！”
<br/>
<br/>        “活该你沦落到这种地步。”
<br/>
<br/>        童筱颖冷冷地撂下话，欲要转身，何明旭却突然说道：“筱颖，我们做个交易如何？”
<br/>
<br/>        “没兴趣。”
<br/>
<br/>        “我知道你是傅夜沉的女人这件事情，你也没兴趣吗？毕竟，当年傅大少爷可是当着我和若琴的面跟你秀恩爱。如果我把这件事情告诉八爷……”何明旭勾了勾嘴角。
<br/>
<br/>        童筱颖回头看着何明旭，嗤之以鼻地笑了笑：“你还真是被关在这里关了三年，成了井底之蛙了。连我嫁给了傅家的二少爷，都不知道。就算你不知道，林若琴应该知道吧！毕竟，她是林贵叔的女儿。”
<br/>
<br/>        “……”听到这里，何明旭不禁皱起了眉头，随即，他又想了另外一招，“我拿马智杰的尸首，跟你换一个人情。你替我向十爷说些好话，让他跟冒先生求个情，保我一条命。”
<br/>
<br/>        他只说保他，没有提及林若琴。
<br/>
<br/>        童筱颖笑容渐敛，冷漠地白了一眼，懒得跟何明旭废话，径直离开。
<br/>
<br/>        原来，何明旭也怕死啊！
<br/>
<br/>        是，谁都怕死。
<br/>
<br/>        她也怕。
<br/>
<br/>        童筱颖走到四爷的房门前，叩响了房门，因为房门是虚掩的，里面传来一声“进来”，她便直接推门而入。
<br/>
<br/>        “把门反锁了。”
<br/>
<br/>        她刚踏入门口，四爷便命令道。
<br/>
<br/>        童筱颖照做，转身关门并反锁，只是这锁她刚落下，身后便传来一股清冽的压迫感。
<br/>
<br/>        下一秒，有个硬物抵在了她的后脑勺上，吓得她一动也不敢动。
