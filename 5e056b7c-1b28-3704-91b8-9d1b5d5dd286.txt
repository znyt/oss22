　　“火舞。”
<br/>　　顾怜星轻呼了一声，搀扶起公孙火舞。
<br/>　　公孙火舞只是被吓着了，并无大碍。
<br/>　　顾怜星看向萧逸，略带不悦之色，道，“萧逸公子，可以开始了。”
<br/>　　“你是修习星光一道的武者，只要实力足够，打赢我天星府武者即可。”
<br/>　　“可以。”萧逸点了点头，脸色淡漠如初，指尖一道星光凝聚而出。
<br/>　　面前，那圣王境五重修为的天星府武者闻言，脸色霎时一白。
<br/>　　“不…不是，少府主…”这个天星府武者，脸庞抽了抽，脸色怪异地看着顾怜星。
<br/>　　“怎么？”顾怜星皱着眉。
<br/>　　这个天星府武者脸色一苦，道，“传闻萧逸小…公子，出手狠辣，与他对战，非死即残。”
<br/>　　“我这点微末修为，这…”
<br/>　　这个天星府武者支支吾吾。
<br/>　　顾怜星反应了过来，点了点头。
<br/>　　随后，也只能尴尬一笑，“萧逸公子的实力，就不用证明了。”
<br/>　　“也行。”萧逸点点头，散去了指间星光。
<br/>　　……
<br/>　　半个时辰后。
<br/>　　在场的一众武者，陆陆续续都与这天星府武者交手完毕。
<br/>　　其中，九成以上武者都证明了自己的实力。
<br/>　　剩下的百来个武者，摇了摇头，也只能自叹实力不足，就此离去。
<br/>　　随后，天星府武者，带着一众武者，进入天星险地之内。
<br/>　　前方，天星府武者在前带路。
<br/>　　公孙火舞与顾怜星并肩而行，脚步有些缓慢。
<br/>　　公孙火舞的脸色看起来，此刻竟比顾怜星还要惨白。
<br/>　　虽看起来无异，但还是不时能看到她的身躯微微颤抖一分。
<br/>　　一旁，西门孤夜脸色凛然，“好恐怖的杀意。”
<br/>　　“这家伙的传闻，难道都是真的？”
<br/>　　“可恶。”公孙火舞煞白的脸色下，咬了咬牙，“本姑娘不会就此罢休的。”
<br/>　　“我劝你不要冲动。”一旁女子低声道。
<br/>　　“那个家伙的战绩，比你想象中的还要可怕。”
<br/>　　“若真的发难，这里的天星府强者，怕也制不住他。”
<br/>　　“若非如此，你以为孤夜真的怕了他吗？”
<br/>　　“我和孤夜倒不太惧他，可你和怜星，绝不是他的对手。”
<br/>　　女子凝重地看了公孙火舞一眼。
<br/>　　公孙火舞低沉地哼了一声。
<br/>　　西门孤夜脸色肃然，心头暗暗惊骇，“一个眼神便能让公孙火舞瘫坐在地，这家伙若真的出手，实力该强到什么地步？”
<br/>　　一旁女子看着西门孤夜的凝重脸色，轻声道，“别担心，我已经暗中传息回天星府。”
<br/>　　“要不了多久，天星府内的长老们会亲自前来的。”
<br/>　　西门孤夜闻言，点了点头。
<br/>　　公孙火舞闻言，得意一笑。
<br/>　　顾怜星，则皱起了眉头，“月姐姐此举会不会小题大做了？”
<br/>　　“我感觉这萧逸公子，倒不是什么坏人，只要不惹他，他也懒得理我们。”
<br/>　　“哼。”女子冷哼一声，“怜星，你入世不深，单纯了些。”
<br/>　　“那等小贼，哪里能以常理视之。”
<br/>　　前方，是天星府的一众武者，以及顾怜星、公孙火舞等人。
<br/>　　后方，则是浩浩荡荡一众武者。
<br/>　　萧逸跟在一众武者中，脸色一如既往的淡漠。
<br/>　　他不知道前方天星府武者中顾怜星等人的对话，但即便知道了，也懒得理会。
<br/>　　他只是确实想看看这天星禁地，到底有何玄奥之处。
<br/>　　据他所知，中域十八府，各自势力范围的地域内，都会存在一古老险地。
<br/>　　而这古老险地中，也必会存在禁地。
<br/>　　如金光险地内的金光禁地。
<br/>　　天星险地内的天星禁地。
<br/>　　不过，金光禁地内，可是生人勿近。
<br/>　　别说开启，若是进入了，即便是武道大能，也是九死一生的下场。
<br/>　　而这天星禁地，居然区区一群圣王境武者就能进入？
<br/>　　而且带队的还只是几个圣王境巅峰。
<br/>　　不错，天星府的这群武者中，所谓强者，不过是一群圣王境巅峰。
<br/>　　心头带着疑惑，萧逸跟着这一众武者，彻底深入了天星险地。
<br/>　　整整一天后。
<br/>　　众人才来到了险地深处，亦停下了前行的脚步。
<br/>　　天星险地，同样异常辽阔，与金光险地差不多。
<br/>　　哪怕是一众武者全速赶路，仍旧耗费了一天有余才来到这深处。
<br/>　　前方，天星府一行人中，顾怜星忽然踏前一步。
<br/>　　手中，一个令牌凭空而现。
<br/>　　令牌上，明显有一绚烂‘星’字。
<br/>　　毫无疑问，这是天星府的令牌。
<br/>　　“起。”顾怜星低喝一声，手中一道道手印打出。
<br/>　　令牌，则漂浮至半空。
<br/>　　不得不说，这顾怜星果真身体虚弱。
<br/>　　哪怕是一声低喝，可声音中，仍旧柔弱如水。
<br/>　　不多时，一个巨大的洞府，拔地而起。
<br/>　　萧逸认真看着，眼眸凝视着。
<br/>　　整个洞府，星光熠熠，仿佛是从星河中落下的神秘府邸。
<br/>　　整个洞府的大小，在数十里以上。
<br/>　　洞府之上，传来的气息中，古老而厚重。
<br/>　　萧逸甚至确信，哪怕自己全力一击，也休想在这星光洞府上留下半分痕迹。
<br/>　　整个洞府，单单是散发的气息，已让人一阵颤抖。
<br/>　　正常来说，这样的洞府，顾怜星这点微末修为，绝不可能轻易唤出。
<br/>　　如果没有看错的话，真正唤出这个洞府的，是那块令牌。
<br/>　　令牌之上，本就存在禁制，两股气息，从中涌现。
<br/>　　一股，冲天而起，连接天空星光。
<br/>　　一股，拔地而下，接连洞府。
<br/>　　令牌，以自身禁制之力，加上天空星光之力，方唤出了这座沉睡在地底不知多少万里下的神秘洞府。
<br/>　　顾怜星打出的手印，应该只是触发令牌禁制的功效。
<br/>　　轰…轰…轰…
<br/>　　周遭山川河流，震颤不停。
<br/>　　洞府缓缓出现之时，整个辽阔天星险地之内，仿佛都处于振动状态。
<br/>　　数分钟后，震动缓缓消失。
<br/>　　洞府，轰然伫立于所有人眼前。
<br/>　　“嘶。”
<br/>　　当洞府真正出现时，那般宏伟、辉煌、绚烂，足以让在场任何武者为之惊叹。
<br/>　　萧逸同样为之惊叹。
<br/>　　但他的惊叹，比其他人略微轻些。
<br/>　　因为，他在金光禁地之下，已经见过洞府了。
<br/>　　不错，面前这个洞府，与金光禁地之下的那个洞府，几乎一模一样。
<br/>　　无论是大小还是样貌，几乎一致。
<br/>　　唯一不同的，便是金光禁地的那个洞府，充斥于金光中。
<br/>　　而这个洞府，充斥于星光中。
<br/>　　萧逸的脸色，忽然一变，脑海中，亦陡然闪过一个猜测。
<br/>　　如果说，其它别的各府禁地中，也是如此的话…
<br/>　　如果说，他的猜测是正确的话…
<br/>　　萧逸放出了感知，认真地感知着面前洞府中的禁制阵法气息。
<br/>　　数秒后，脸色一变。
<br/>　　“果然，连阵法气息都一样。”萧逸心头一凛。
<br/>　　如果他的猜测没有错的话，那么，这些禁地之下的洞府禁制阵法，已经根本不能称之为阵法了。
<br/>　　而是…一个局。
