　　楚风眠凝聚出这战龙之爪，速度快到吓人。
<br/>　　他的战龙之爪，是来自与血脉之中的力量，瞬息之间便是就凝聚出来，一招落下。
<br/>　　这年轻男子打出的五轮秘法，还没有爆发出威力来，便是被楚风眠这一招战龙之爪，直接拍碎。
<br/>　　反噬的力量，更是让那年轻男子，都是不由的吐出了一大口精血。
<br/>　　他的眼神充满震惊，仿佛是从来没有想到过，自己居然会败。
<br/>　　他可是北邙学院中都无敌的人物，在北邙学院中，他都足以位列上位，地位极高。
<br/>　　这一招五轮秘术，在北邙学院中都赫赫有名，如今没想到在这偏远的武胜国度，居然会被人击退。
<br/>　　“卑贱的小子，你居然敢破了我的五轮秘术，你死定了！敢跟本少为敌的人，都要死！”
<br/>　　这年轻男子怒喝一声，猛然之间，身上爆发出来的力量更加强横。
<br/>　　“北邙之力！加持我身！”
<br/>　　在他的身躯之中，仿佛是有着一尊种子，突然破碎了，随着这种子的破碎，无穷无尽的力量，加持在了这年轻男子的身上。
<br/>　　他的实力，节节攀升，比起刚刚，更是强大了一倍不止。
<br/>　　猛然之间，年轻男子一拳轰出，对着楚风眠直接砸了过来。
<br/>　　“哦？这倒是一道从未见过的秘术。”
<br/>　　楚风眠看到这年轻男子身上的变化，也是微微有些吃惊，他是万年前的人物，知晓无数万年前的秘术。
<br/>　　但是这年轻男子展现出来的如今这一招，楚风眠却是从未见过，看起来应该便是北邙学院中独特的秘术。
<br/>　　这一招，倒是有着几分独到之处。
<br/>　　“卑贱的小子！你逼得本少动用圣种的力量！本少今日便是要杀了你，将你挫骨扬灰！”
<br/>　　年轻男子疯狂怒喝一声道，猛然之间，整个天际都是陷入了一片黑暗之中。
<br/>　　这边是他的领域，无尽黑暗的领域，在这无尽黑暗之中，仿佛是有着无数生灵，都在其中栖息。
<br/>　　“封锁！”
<br/>　　随着年轻男子的怒喝，周围的黑暗之中，突然出现了无数锁链，一道道的向着楚风眠封锁而来，要将给团团封锁住，令楚风眠有力量都无法动用出来。
<br/>　　“神圣之矛！”
<br/>　　年轻男子手心一动，一根长矛便是直接被凝聚出来，这长矛，正是他身躯中所蕴含的灵力，凝聚而出。
<br/>　　看起来，似乎像是一尊灵器的虚影。
<br/>　　“天级灵器的虚影？”
<br/>　　楚风眠看到那年轻男子手中的神圣之矛，心中冷笑道极点，他大手一挥，一座石碑，顿时出现在了楚风眠的手中。
<br/>　　“给我碎！”
<br/>　　丰饶碑的力量，顿时催动到了极致，以现在楚风眠的实力，已经是足以催动丰饶碑近乎全力的效果。
<br/>　　丰饶碑一出，无尽灵力直接是压迫而下，顿时轰击了过去，将这年轻男子手中的神圣之矛，都给直接打碎。
<br/>　　“天级灵器？”
<br/>　　看到楚风眠手中的丰饶碑，年轻男子的眼神中都是不可置信。
<br/>　　这天级灵器，他都没有，在北邙学院中都是极为罕见，只有一些强大的长老，才拥有。
<br/>　　没想到楚风眠居然是有着一尊真正的天级灵器。
<br/>　　这丰饶碑的力量，不单单是打破了神圣之矛，更是直接将楚风眠身边的无数锁链，都给一下子震破开来，令他的身形一下子自由了出来。
<br/>　　“你如今的力量，应该便是来自于你身躯中的那枚种子吧！”
<br/>　　楚风眠看着年轻男子，冷喝一声。
<br/>　　“剥夺！”
<br/>　　一团灵力凝聚出来，便是化为了一道手臂，直接向着那年轻男子抓了过去。
<br/>　　他要将这种子抓出来，一探究竟，北邙学院是万年来新崛起的势力，楚风眠对此都没有任何了解。
<br/>　　如今正是要从这种子上，来看看北邙学院的功法，知己知彼才能百战百胜。
<br/>　　“休想！你敢剥夺圣种！这是对我们北邙学院的亵渎！卑贱的小子！你会受到我们北邙学院无穷无尽的追杀的！”
<br/>　　年轻男子看着楚风眠的举动，脸色大变道，大声的威胁了起来。
<br/>　　“闭嘴！”
<br/>　　楚风眠冷眼一横，直接对着那年轻男子一巴掌打了过去，打在了他的脸上。
<br/>　　“啪！”
<br/>　　这一声清脆声音，令在场寂静无比，所有人的脸色都是惊恐到极点。
<br/>　　楚风眠居然是敢打这年轻男子，这年轻男子，可是北邙学院的使者，这打的可不止是他的脸，更是北邙学院的脸。
<br/>　　“小畜生！你敢打我……？”
<br/>　　年轻男子都是一愣，随后怒火爆发出来，还从未有人敢打过他耳光，这简直是对他的无上羞辱，令他气愤到极点。
<br/>　　“打你了又如何？”
<br/>　　楚风眠最一撇，又是一把巴掌打了过去，打在了另一边的脸上，令这年轻男子的脸，一下子肿了起来，好像是如同猪头一般。
<br/>　　“不止是要打你！我今日还要降伏了你！给我跪下！”
<br/>　　楚风眠冷喝一声，身上的战龙龙威，彻底的爆发了出来，压迫在了这年轻男子的身上。
<br/>　　无穷无尽的威压，压在这年轻男子的身上，令他脸色涨的通红，甚至是微微有些发紫。
<br/>　　他的一身灵力，都被催动了起来，要来抵挡着楚风眠的威压。
<br/>　　“该死！本少可是北邙学院的内门弟子，即将要晋升精英弟子的人！地位尊高，血脉高贵无比！怎么会跪你这样一个破落国度的野小子！”
<br/>　　年轻男子大声怒喝到。
<br/>　　“我永不败！”
<br/>　　在这年轻男子的身上，居然是又有着一股力量从他的身躯深处，爆发了出来，要隐隐约约跟楚风眠的战龙龙威抗衡。
<br/>　　“哦？看来这种子的力量便是你现在的力量源泉，那好，剥夺！我看你还能如何反抗！”
<br/>　　楚风眠冷笑一声，他自然看得出来，这年轻男子其实早已经是没有力量，全凭借这身躯中的圣种的力量来苦苦支撑。
<br/>　　这圣种，仿佛是一尊大能种在他身躯之中的，为他源源不断的提供力量。
<br/>　　如今楚风眠大手一挥，直接对着这年轻男子抓了过去，要将在藏在他身躯身处的圣种给剥夺出来。
