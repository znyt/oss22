<br/>　　周围的上百名圣院宗圣子，都恭敬的站在这男子的身旁，显现出了臣服。

<br/>　　楚风眠看向这男子的刹那之间，就感觉到了一股极大的压力，笼罩而来。
<br/>　　这一股压力，并非是男子故意压迫而来的，而是在他的身上，自然而然散发而出的气息。
<br/>　　霸者武道，武道之中最为霸道的一种，要让天下臣服，所向无敌，无人可以与之并肩。

<br/>　　修行这霸者武道，看似容易，但是一生不败，谈何容易？几乎不可能，只有天才之中的天才，才足以达到一生不败的可能。
<br/>　　这种霸者武道，楚风眠也曾经知道许多的强者修行过，但是无一例外，都失败了，没有人可以在这一条道路上，走到极致。

<br/>　　这一点便是无比可怕了，也造就了此人如今这强横的武道。

<br/>　　炎撼世也看到了人群中心的那一尊尊贵男子，看了一眼突然脸色大变对着楚风眠说道。

<br/>　　起码是在上一次圣地之中相见的时候，这寒夜的境界还依然是八劫古帝，距离突破九劫古帝还相差甚远。

<br/>　　毕竟天下机缘，人人都有可能得到。

<br/>　　楚风眠目光闪烁，嘴角露出了几分冰冷笑容。
