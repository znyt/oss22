
<br/>　　<br/>　　    小摊，撸串，啤酒。<br/>　　<br/>　　    常文书从包里拿出一叠纸推到他面前，说道：“一共七个人回了信，都是京城这方面最资深的专家，其中有个古语言教授还亲自给我打了电话，询问我这些文字是从哪里找到的，我就说是一个朋友收的古董上面印着的，托我找人问问写的是什么，他说这些文字存在的年代应该是三千五到四千年前左右，算是甲骨文和后期钟鼎文的一种吧，也叫甲骨卜辞，不过存世的已经很少了，非常冷僻，他就算是这方面的专家，我交给他的那十来个文字，也用了六七天最后才翻译出来，但是他说有可能意思还会有出入，毕竟翻译这种文字的话，最好是用全篇来翻译才行，这样才能通过前后的意思来判断出到底是什么字”<br/>　　<br/>　　    “这老头也够精的了，他是看出来有可能是一整篇的，才这么和你说的”王惊蛰皱了皱眉，擦了下手上的油渍，然后拿起那叠纸翻看了起来：“等我全都捋完了的，实在不行就找他从头再看一遍吧”<br/>　　<br/>　　    “嗯，他也说了随时找他随时好使，人家很明确的透露出感兴趣了，我觉得吧这东西如果不是太过主要的话，你又没太看懂，可以跟他商量商量”常文书端着酒瓶子，说道：“来，边吃边看，不耽误事”<br/>　　<br/>　　    “哗啦”王惊蛰翻看着几页纸，上面写着的都是一些古怪和生僻的字眼。<br/>　　<br/>　　    “提红，班鳌”<br/>　　<br/>　　    “也绿草”<br/>　　<br/>　　    “寒牙藤……”<br/>　　<br/>　　    王惊蛰看了几页，手里的肉串就放下了，心里顿时“咯噔”了一下，这写翻译过来的东西已经跟他的猜测越来越近了，再往后到最后两三页的时候，这几个翻译的词让他手瞬间就颤了几下。<br/>　　<br/>　　    “温火六息”<br/>　　<br/>　　    “急火圭表三度”<br/>　　<br/>　　    “石鼎没水……”<br/>　　<br/>　　    看到最后一页几个词的时候，王惊蛰就舔了舔嘴唇，手指都不自觉的捏紧了：“丹成服三，养魂一甲”<br/>　　<br/>　　    王惊蛰呼吸一下子就急促起来，胸膛里一颗心犹如小鹿乱撞，砰砰跳个不停。<br/>　　<br/>　　    几天前在拿到这个胛骨的时候，王惊蛰就隐约猜测，这有可能是一张道家丹方，不过当时还不太确信，因为道家炼丹一术的精髓早就已经失传了，现在剩下的也许连冰山一角都算不上。<br/>　　<br/>　　    玄门五术里，胡胡所在的山字脉中就有炼制丹药一项，只是如今能炼出来的顶多也就是延年益寿，保养滋补一类的丹药罢了，再高深点的则就失传了。<br/>　　<br/>　　    有人可能会觉得道家炼丹术神乎其神，属于谣传，其实说白了一点都不神，如果通俗易懂的去解释，现今有一补药其实就是炼丹的一种，叫六味地黄丸，而如今中医开的很多药方，往前延伸的话，可能九成九都属于炼丹的一种方式，所以在中医和道门中始终都有个争议，或者不太确定的说法，那就是华夏中医有很大的可能都是源自于道家炼丹术的，不过就是没办法证明罢了。<br/>　　<br/>　　    而古往今来，从秦朝开始再到清朝末年，关于笃信炼丹能长生不老的皇帝大有人在，在最初期最典型的莫过于秦始皇差遣徐福驾船出海为其寻找炼制长生丹药的配方，却导致虚徐福一去不归，而到了唐代那时候则是炼丹的全盛时期了，几乎历代皇帝都热衷于此，哪怕是一代女皇武则天和李世民了，这些皇帝都想着要长生不死，然后还被祸害的不清，其中唐太宗，宪宗，穆宗还有晚唐的武宗等几个皇帝都是服丹药而身亡的，这还不算另外几个中了毒且没死的皇帝。<br/>　　<br/>　　    丹药一道，玄归玄，神归神，可几千年来的历史痕迹证明，道家炼丹术确确实实是存在的，但是不知道为何久而久之下，就全都失传了，如今剩下的这点皮毛都可以忽略不计了。<br/>　　<br/>　　    再说王惊蛰看着翻译出来的几页文字，寥寥几十个忽然间给他打开了一扇满怀憧憬的大门，前面译出来可能是一些草药的名字，还有制丹的方式，特别是最后一页上所写的那两句，丹成服三，养魂一甲，他估计这胛骨上记载的丹药，有可能是滋养人三魂七魄的，如果真是这样的话，他要是能捅咕出来，肯定是受益匪浅啊。<br/>　　<br/>　　    “在催催剩下那几位，尽快把其余的那点翻译出来吧……”王惊蛰收起桌子上的几页纸跟常文书说道。<br/>　　<br/>　　    “呵呵，你知道是啥玩意了啊？”常文书斜了着眼睛问道。<br/>　　<br/>　　    “差不多吧，八九不离十能猜的出来了，不过还得最后再确定一下”<br/>　　<br/>　　    “其实你不说我也看出来了，这好像是什么配方，那个也绿草我听过是味中药”常文书挠了挠鼻子，凑到王惊蛰面前，脸上写着我也懂的表情，小声说道：“这东西一看就很多年了，我听说以前的皇帝都喜欢服用壮阳的药，要不然后宫佳丽他们都拿啥伺候啊，夜夜七次郎，拿啥一路坚挺啊，是不是这个道理？哥们，你就跟我说吧，这到底壮不壮阳……”<br/>　　<br/>　　    王惊蛰崩溃的说道：“咋的，你不行了啊？”<br/>　　<br/>　　    常文书顿时跟踩了尾巴似的，摇头说道：“别闹，啥不行啊，主要是我觉得吧武功在高点毕竟不是什么坏事，真要是壮阳的，你捣鼓出来后，我先给你试试药呗？”<br/>　　<br/>　　    “哎呀，行了，行了，你想哪去了，配方是配方但跟床上那点事没关系”<br/>　　<br/>　　    “不是，你别诓我，我可认真了哈……”<br/>　　<br/>　　    “这没准都几千年的玩意儿了，你要真不信到时候你就吃吃看，要不怕给自己吃死了你尽管下嘴，好像我还骗你似的，我看你啊还真就是不行了”<br/>　　<br/>　　    常文书急眼了：“你要是这么说的话，那今天我说什么都得带你出去溜一溜了，用实践来证明下”<br/>　　<br/>　　    “滚蛋吧，吃完没有，吃完了赶紧走，剩下的那点你快点帮我催催吧”王惊蛰拍着他肩膀说道：“其实想坚持长久点，也不难，有现成的人可以问问”<br/>　　<br/>　　    “哎？问谁？”<br/>　　<br/>　　    “你问国足啊，是怎么保持九十分钟不那个啥的，呵呵，一个半小时还不够你嗨了？”
