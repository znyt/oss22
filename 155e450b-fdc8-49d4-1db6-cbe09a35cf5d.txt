　　第163章
<br/>　　几十人几十人抱团在一起，丢盔弃甲，表情麻木，视线放到最高处看去，沿着蜿蜒的官道，败兵如蚁。
<br/>　　蛤蟆寨，聚义厅。
<br/>　　程大雷坐着代表寨主的太师椅上，左手边徐神机、黄三元、刘关张、秦蛮、赵子龙……右手边高飞虎、高飞豹、小白狼、熊大、熊二等原先五十多家山寨的头目。
<br/>　　乌云盖顶，沉重得令人喘不过气来。
<br/>　　“大当家，咱们往那里走？”高飞虎。
<br/>　　“走？”程大雷重复这个字，从他的表情上看不出他在想什么。
<br/>　　“我说句不太中听的话，大当家的为人我们都是佩服的，没有大当家不可能有咱们的今天。但现在戎族过来了，各家兄弟只有散伙，日后若都能活着，兄弟们聚在一起，咱们还以程当家为首。”
<br/>　　根据黄三元和赵子龙带回来都消息，北蛮部已经攻破黑石城，此刻定在屠城。黑石城里无论老弱，必然是血流成河，浮尸满野。
<br/>　　程大雷若有圣人兼济天下的胸怀，当要赶走戎族，解苍生与水火。
<br/>　　可惜他没有，他终究只是个普通人。
<br/>　　即便是个普通人，在想到黑石城会发生的事情时，程大雷还是有些不开心的。
<br/>　　问题是，北蛮部这次究竟要做什么？
<br/>　　是像从前无数次一样，杀一些人，抢一些物资和女人就走，还是以黑石城为开始，掀开劫掠整个帝国的序幕？
<br/>　　黑石城，究竟是结束，或者仅仅是个开始？
<br/>　　若黑石城仅仅是个开始，青牛山便是戎族的必经之地。
<br/>　　北蛮部能一日破五万幽州兵，半日破黑石城，何况一座小小的蛤蟆寨。
<br/>　　所以大家的第一个念头就是逃，五千人的山寨做鸟兽散，个人顾个人，向帝国的腹地逃，逃得越远越好，永见不到戎族才好。
<br/>　　聚义厅开始乱了，帝国子民早已到谈戎色变的地步，大家乱哄哄一片，想的是怎样逃，是走陆路，往冀州并州逃，还是走海路，去那传说中的海外孤岛，或去扬州徐州这样远离边境的地方。
<br/>　　程大雷没有说话，因为他此刻心里也没有主意。
<br/>　　程大雷心里没有多少为国为民的情怀，他即不热爱帝国，对戎族也没有多少仇恨。当然，身为一个四等不入的山贼，帝国也从来没有将他当作子民。
<br/>　　可也因为这样，程大雷对戎族并没有多少畏惧心理，大家都谈戎色变，望风而逃，而程大雷心里则不是这样想。
<br/>　　他心里的想法是：戎族不满万，满万真的不可敌么？
<br/>　　良久，程大雷轻咳一声，纷乱的聚义厅渐渐安静下来，大家的视线落在程大雷身上。
<br/>　　“逃，咱们又往那里逃？”程大雷问。
<br/>　　“逃，自然是往南走，避开戎族，咱们占个山头，依旧作威作福。”
<br/>　　“或者从海路走也可以，我听说海上有片孤岛，只要入了海，戎族绝对奈何不了咱们？”
<br/>　　个人都出着个人的注意，眼看大厅内又要乱起来。
<br/>　　程大雷挥挥手：“咱们山寨有五千余人，老弱妇孺就有一多半，真正能动手的不过两千几人而已。如果戎族占了黑石城还不满足，又要南下，那我们两条腿跑得过戎族的铁蹄嘛。当然，兄弟们都是有本事的人，肯定有人能在这次大乱中活下来，可那些孩子和老人也能活下来么？”
<br/>　　众人沉默下来，所谓兵败如山倒，逃兵不如狗，真如果分散逃跑，谁能活下来，谁不能活下来，就得靠拼脸撞大运了。
<br/>　　“大当家，你有什么主意，与大伙说了吧。”
<br/>　　程大雷摇摇头：“我没有主意，我要有主意，我就不和你们废话了。”
<br/>　　只闻戎族强悍，究竟有多强悍程大雷却并不了解。他现在介于心里有些犯怯而又想试试的边缘。
<br/>　　大厅又乱下来，这些山贼其实并没有多少主意。听到程大雷的分析后，都已束手无策。
<br/>　　“如果……我说的是如果……”程大雷问：“如果我们不逃的话，可不可行？”
<br/>　　“大当家的意思是……”刘悲看穿了程大雷的用意：“想要和戎族战一场？”
<br/>　　一片哗然，当程大雷心中的念头稍微露出点尾巴后，顿时吓了众山贼一跳。
<br/>　　“这根本不行！”有人高呼道：“杨龙停五万大军，一日就被戎族攻破，咱们山寨才五千人，能打的不过两千人。”
<br/>　　“咱们能跑，为何要和戎族打！”
<br/>　　“大当家是要我们都送死么！”
<br/>　　“都住了吧。”程大雷忽然起身：“现在的我还是山寨的大当家，现在就不给我面子，是不是忒早了些。”
<br/>　　人群安静下来，毕竟程大雷积威还在。
<br/>　　程大雷压压双手：“我也没说一定要守，或者一定要逃，我不是在与大家商量嘛。逃是个办法，可用后背面对敌人的铁骑，我敢说今日山寨五千人，能活下来一两千就不错了。而如果守的话……”
<br/>　　程大雷略顿了顿，手挥过似乎想将整座山寨笼罩。
<br/>　　“第一，山寨地形易守难攻，只要把住寨门，任他千军万马都攻不破；第二，山地不宜骑兵行军，戎族的优势将被大大削弱；第三，我们山寨存粮无数，撑个一年半载是绝没有问题的，可戎族没有后勤粮草支援，撑不动，他们自然就会撤兵。”
<br/>　　程大雷数到第三条，道：“当然如果死守的话，也会有很多人牺牲，包括我，包括各家兄弟，但我觉得有以上三条，未必守不住。我也并非大权独揽，今日是关系五千多弟兄的生死，自然是各家兄弟一起商议。我问一句，愿意战的把手举起来？”
<br/>　　声音落地，众人面面相觑，互相打量着彼此，目光闪烁却也不知道在想些什么。
<br/>　　然后，坐满人的大厅，只有一支手举了起来，若孤独的旗杆。
<br/>　　是林少羽。
<br/>　　这个要成为大侠的男人，终究没令程大雷失望。可只有他一人举手，却也显得那样可怜。
<br/>　　即便徐神机、秦蛮等与程大雷亲近的人，也无法鼓起勇气，放手与戎族一战。
<br/>　　戎族，就那么可怕么？
<br/>　　越是这样，偏偏越激起程大雷的好奇心。
<br/>　　好奇，有时候也是好胜。
<br/>
