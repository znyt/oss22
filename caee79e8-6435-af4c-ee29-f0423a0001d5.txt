        “她是傅夜沉的新欢兼女秘书！”八爷不痛不痒地回答。
<br/>
<br/>        “傅夜沉不是死了吗？”童筱颖不以为然道。
<br/>
<br/>        八爷一边把玩着手里的枪，一边咧嘴道：“哼，我也以为他死了，可是他没死。冒先生的线人告诉我，傅夜沉和阿十里外接应，把雅珊从安防局的暗牢里救了出来。傅夜沉背叛了安防局，现在已经成了安防局的通缉对象。同样，冒先生现在也在找傅夜沉，抓这个女人来，就是为了引傅夜沉出来。而抓你，则是为了引阿十出来。”
<br/>
<br/>        “十爷跟你们是一边的，抓我引十爷，你是不是有些画蛇添足？”童筱颖不以为然地说。
<br/>
<br/>        八爷接着说道：“因为，阿十现在背叛了冒先生，和傅夜沉站在了一起。他俩到底是叔侄关系，自家人站在一起也很正常。可是，他们两若是联合，对冒先生来说就是比安防局还要大的威胁。抓了你两，我们布置好了陷阱，就等着他俩上钩。冒先生最痛恨的就是叛徒，这回，我也帮不了阿十了。不过，你放心，阿十和傅夜沉死后，我会让你给他们陪葬，算是看在我和阿十曾经做‘兄弟’的份上。”
<br/>
<br/>        “原来八爷也讲江湖义气啊！”童筱颖淡然。
<br/>
<br/>        听到这里，落以柔顿时不安分地“唔唔唔”地激动起来。
<br/>
<br/>        八爷见她似乎是有话要说，伸手撕掉了落以柔嘴上的胶布。
<br/>
<br/>        与此同时，直到这一刻，童筱颖才明白，原来是自己将傅夜沉逼到众叛亲离的这一步。
<br/>
<br/>        落以柔的嘴唇被胶布粘破了皮，一丝鲜血顺着她的嘴角滑落，她吃痛地皱着眉头，下一秒，就开始指控童筱颖：“我只不过是傅总的秘书，而她童筱颖才是傅夜沉真正的女人！”
<br/>
<br/>        “你说什么？”八爷诧异。
<br/>
<br/>        落以柔恶狠狠地看着童筱颖，口不择言地接着说道：“她童筱颖，才是傅夜沉真正的女人！而且还是傅夜沉的合法妻子！你绑架我，压根就引不出傅夜沉。只有她童筱颖，才有这个本事！”
<br/>
<br/>        童筱颖听着落以柔的这番指控，心止如水，脸上没有任何表情。
<br/>
<br/>        八爷则不可思议地看着童筱颖：“她说的是真的？”
<br/>
<br/>        “是！”童筱颖毫不畏惧地坦白，“所以，你可以把她给放了。”
<br/>
<br/>        “你不是说你是傅夜沉的弟媳吗？”八爷还是难以置信。
<br/>
<br/>        童筱颖冷冷一笑：“以前是，现在不是。”
<br/>
<br/>        “你这个女人，耍我！”八爷顿时恼火，又从腰间掏出手枪，抵在了童筱颖的额头上。
<br/>
<br/>        童筱颖仍旧十分淡定，似乎对于他举枪要杀人的样子已经见怪不怪了：“你抓我一个，就可以威胁到傅夜沉和十爷了。所以，你没必要把另一个没用的女人也牵扯进来。而且，一个没用的女人，对你八爷来说，也是个麻烦。”
<br/>
<br/>        听到童筱颖这番话时，落以柔对童筱颖倒是刮目相看。
<br/>
<br/>        她完全没想到，童筱颖会以德报怨。
<br/>
<br/>        明明是她落以柔出卖了她童筱颖，将她陷于险境之中，她却在想法子救她，想独自当人质。
<br/>
<br/>        落以柔此刻死死地盯着童筱颖，话中有话地接着问道：“童筱颖，你是真的爱傅夜沉吗？你若爱他，那么他的命是不是比你的命更重要？”
<br/>
<br/>        “……”童筱颖冷静地对上落以柔的眼睛，终于明白，她和落以柔的区别是什么了。
<br/>
<br/>        “我早该想到你跟傅夜沉有一腿才对！”八爷忽然勾唇一笑，依旧用手机指着童筱颖，对坐在一旁的手下命令道，“打开车门，把这个没用的女人扔下去。”
<br/>
<br/>        “是！”戴着墨镜的男人立即将车门打开，二话不说，就将落以柔推下了车。
<br/>
<br/>        童筱颖只听到落以柔“啊”的惨叫声，虽然落以柔会被摔痛，但总比被挟持着等着被凌迟要好。
<br/>
<br/>        “被推下车的那个女人，是死是活，就听天由命了。怎么样，我卖了你一个人情，应了你心里的想法，放过了那个女人一命。你打算怎么谢我？”八爷将枪口从童筱颖的额头上往下挪动，直接来到了她的衣领处插进去，抵在她的锁骨上。
<br/>
<br/>        童筱颖临危不乱地问道：“你为什么恨傅夜沉？”
<br/>
<br/>        “因为，我深爱的女人，是因他而死。”八爷剑眉微蹙，假面眼镜下两个眼洞里黑白分明的眼睛，透着一丝寒意，“但是，我答应过她，不会杀了他。既然杀不了他，那么就跟他玩玩喽！制造点小暴乱，让他当英雄玩玩，顺便将他的生死交给老天爷来决定。哪知道，他每次都那么命大。”
