　　第689章
<br/>　　天子的权力在万里之内，十步之外。不过，目前李乐天的权力已不可能抵达万里，面对诸侯割据的帝国，充其量也就是京州这一亩三分地。而且还是有名无实。
<br/>　　至于十步之内，李乐天也没多少权力。大内不知安插了多少眼睛，盯着自己的一举一动。李乐天身边连个能商议事情的谋士都无。
<br/>　　现在，他越来越能体会明帝的无奈，在帝国风雨飘摇的关头，还能维持帝国的平稳，明帝其实并不容易。
<br/>　　李乐天不是一个昏君，他是实实在在想做些事的。但他能动用的力量其实微乎其微，想做些事也是有心无力。
<br/>　　必须得想个破局的法子了。李乐天手指敲着桌面，他没有可信任的人，必须沉下心来自己琢磨。
<br/>　　长此以往下去，自己当真就成了摆设，这李家天下快成了姓崔的帝国。
<br/>　　该从何处破局……
<br/>　　李乐天脑海中浮现程大雷的名字。他当然对程大雷恼恨到极致，一个山贼委实有些不知天高地厚。可现在并非意气用事的时候，有句话说得好：敌人的敌人就是朋友。
<br/>　　或许，自己该从程大雷处借力，打破长安的僵局，借此削弱相府的势力。
<br/>　　他顺着这个思路想下去，陷入沉思……这次并不是没有好消息到来。
<br/>　　凉州一直是相府在布局，偏偏生出一个程大雷，竟将相府扶植的宋伯康赶了出去。更值得高兴的是，有白玉美人之称的崔白玉落到了程大雷手中。
<br/>　　当然，这件事对李乐天没多大好处。但相府吃了亏，终究是一件值得高兴的事。这帝国天子，在无能为力时也有些恶趣味。
<br/>　　却也并非不能从程大雷处借力，他打着父王的名号，自己便索性顺水推舟，封他做凉州王。
<br/>　　但既然给了程大雷名份，程大雷就必须为此付出代价。李乐天沉下心来，想着可以从程大雷处要点什么……
<br/>　　……
<br/>　　不止李乐天，相府也已经得到了消息。
<br/>　　床榻之上，一个干瘦的老人伸出手，无力的在空中挥了挥。
<br/>　　崔云间立在一旁，脸色也不太好看。
<br/>　　事实上，崔相的情况也不如李乐天想得那么好。这个老人权势熏天，挥挥手就是满天风雨，一个个政敌在他面前倒下，如今已没有谁有资格立在他面前。然而，有一个敌人是他敌不过的，那便是岁月。
<br/>　　他已经老了，已经老到该安排后事的时候了。
<br/>　　“天子不甘心被我崔家操纵，迟早有撕破脸的时候，呵，李家果真是没有孬种呐。”
<br/>　　崔云间认真听着，但心中还是有些不服气，道：“现在朝堂上下都是我崔家的人，他即便不答应，又能翻出什么波浪？”
<br/>　　老人无力的摇摇头：“你不懂，那些世家豪门易聚也易散，永远不要将他们当做自己的力量，力量这种东西永远要握在自己东西。有朝一日，这崔家总要交到你手中，你记得我两句话。第一，永远要学会把利益分给别人，不可太贪婪。第二，崔家可以不赢，但永远不能输。”
<br/>　　崔家有崔家的难处，他们最大的难处便是后继无人。在崔相之后，第二代，第三代年轻人中，都没有过分出类拔萃的存在。虽然某些子弟也被捧成文武全才，但其实他们肚子里几斤几两，崔相还是清楚的。
<br/>　　他今夜这番话就是表明，日后崔家是崔云间做主。也顺带帮忙为他指明今后的路。
<br/>　　崔云间沉默片刻，道：“父亲，儿有一事不明，还请父亲解惑：我崔家真做不了那张椅子么？”
<br/>　　崔相无可奈何的摇摇头：“永远不要这么想，我崔家能够历经百年不衰，凭的就是我们从不在一方全部下注。日后这天下说不好是谁家的，我们要早早做好打算，保持我崔家的基业不倒。”
<br/>　　世事混沌，如崔相这种老狐狸也看不清楚太多。他早已算定，李家要丢掉天下，但偏偏生出一个李乐天与李行哉，又让李家多了几分气数。江南一个李星，估计也是难成大事。
<br/>　　这些都在崔相的眼中，他看不穿的偏偏是程大雷。这个人忽然冒出来，在极不可能的情况下，挣出一片天地。
<br/>　　崔相隐隐觉得，程大雷背后有一股莫名的力量，但这力量究竟来自哪里，他，看不穿。
<br/>　　“父亲，白玉还在程大雷手中，我们要不要与他聊聊，将小妹赎出来？”崔云间道。
<br/>　　崔相摇了摇头，有些话他并没有同崔云间说，毕竟说了他也未必懂。程大雷是一个未知数，日后说不得会走到哪一步，将崔白玉留在他身边，或许，也能为崔家争来几分气数。
<br/>　　滔滔浊世，人在其中不过几朵浮尘碎沫，狡猾如崔相，也只能随波逐流，祸福由天。
<br/>　　程大雷占据凉州后，这件事在帝国造成的风波不小。帝国十八路诸侯，至少，程大雷占了一路，现在也是有头有脸的人物。其他诸侯不免寻思，这山贼出身的恶棍，说不得还会搞出怎样的风雨，大家不得不防。
<br/>　　而这也变相刺激了另外一部分人，大家同是山贼，程大雷也并非三头六臂，他可以做到的事情，大家也可以做到么。
<br/>　　诸人蠢蠢欲动，想着在这场乱世赌局中攥取一部分利益。无形中，程大雷起到了榜样的作用。
<br/>　　东海落凤城，当李行哉得到消息后，不动声色的将情报在烛火上燃尽。
<br/>　　终于，你也迈出了那一步，未来再次相遇，真也许就是在疆场之上了。
<br/>　　李行哉最近发展得不错，有了书院相助后，他手里不缺人手。东海毕竟富饶些，落凤城在这段时间也聚集了不少人，小小一个落凤城已经是一股不容忽视的势力。
<br/>　　很显然，落凤城的发现已经到瓶颈，李行哉要扩大一下自己的地盘了。
<br/>　　他有三条路走，拿下徐州，或者拿下兖州，更或者，南下拿下富饶的江南。
<br/>　　得江南便得天下粮仓，坐南望北，争天下自己便多了几分底气。只不过江南河道纵横，也是极其的不好打。
<br/>　　但徐州、兖州很明显已经无法满足李行哉的胃口。
<br/>(本章完)
