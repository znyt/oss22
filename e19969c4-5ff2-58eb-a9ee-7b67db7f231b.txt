　　萧逸云仍然笑着，把玩着天师牌，心不在焉地说道：“说来我听听。”
<br/>　　叶少阳刚要开口，朝左右看了看，说道：“你还是到我这边来，有些秘密，被别人听到了不雅。”
<br/>　　萧逸云一怔，倒也不怕他耍什么花样，于是走出门外，往边上没人的地方走了走。
<br/>　　叶少阳也走过去，低声说道：“你是北宋年间一史官的儿子，你爹因为不满皇帝让他篡改历史，被皇帝抓起来，然后逼你写，想要羞辱你父亲，结果你也不愿意写，皇帝判你们问斩，你跟你父亲在狱中相约自尽，死后，崔府君看你父子二人刚烈，判你父亲往生贵族，你因为字写的好，被崔府君收为书童，后来提升押司，长住天子殿，你是崔府君第一亲信，我说的对不对？”
<br/>　　萧逸云听他说了这么多，略有惊讶，不过随后淡淡一笑，道：“这些历史，并不是只有一个人知道，我只能说，你打听的挺详细的。”
<br/>　　叶少阳无语，想了一下说道：“你中间受崔府君暗中指点，投生过一次，去人间寻找……”
<br/>　　“住口！”萧逸云突然喝住了他，脸色微变，说道：“这件事你从哪里听来的！”
<br/>　　叶少阳耸了耸肩，“你自己告诉我的。”
<br/>　　“我自己告诉你？”萧逸云冷哼。
<br/>　　叶少阳道：“先不说这个，你就说，我说的这件事，你是做过还是没做过？”
<br/>　　萧逸云沉吟半晌，说道：“这件事，虽然知道的人甚少，但也并非没有人知道……除非你真能说出外人觉不知道的事情，否则我只会更怀疑你别有用心而来。”
<br/>　　叶少阳真心感到无语，挠着后脑勺，想了半天，突然眼前一亮，说道：“有件事情，你跟我说，整个阴司都没人知道，不过我不确定是你骗我的还是真的。”
<br/>　　萧逸云昂首看着他，意思让他说下去。
<br/>　　叶少阳道：“你曾经暗恋过一个姑娘，不，应该说是阴司，速报司的一个提辖使，叫媚儿，你跟她一起出过公差，去当监军，你喜欢过她，但是你没有表白，她就转世投胎，这件事你就烂在肚子里，再没跟别人说过……”
<br/>　　在他开口说的时候，萧逸云还是一脸的不信任，听他说到后面，一张脸完全变了，怔了半天，突然一伸手抓住他的领子，冷冷说道：“这件事不可能有人知道，你是怎么知道的！”
<br/>　　“还是那句话，是你自己告诉我的。”叶少阳吐出一口气，“当时你到人间，我带你去上网，回去之后，你大到我那里睡，然后我提议喝酒，你找人从阴司给你送了酒，还逼我魂魄出窍，跟你一起喝，最后我们两个都喝醉了，你跟我说了这件事……你当时还说，那姑娘十分高冷，对你不大搭理……”
<br/>　　“够了！”萧逸云睁大眼睛看着他，厉声道：“你到底是谁！”
<br/>　　“叶少阳。茅山天师。”
<br/>　　“我并不认识你，没听说过你的名字，也没见过你。”
<br/>　　“我从一百年后来，一百年后，你跟我是兄弟，我是茅山第三十九代掌教，人间第一天师，大帝亲自赐我牌匾，我一帮门人在阴间开立府衙，叫阴阳司，为阴司第七十三司，专管人间法术界解决不了的事，我一门人在你天子殿当上了副押司……你还想知道什么？”
<br/>　　叶少阳连珠炮似的将这些事情一口气讲了出来，萧逸云听完之后，整个人完全呆住了，望着他，摇了摇头。
<br/>　　“神经病！你不是疯了吧！一百年后的人，怎么会出现在我面前，穿越时空吗？”
<br/>　　叶少阳道：“你答对了，我真的穿越时空了。”
<br/>　　“不可能！”萧逸云不假思索，“没人能穿越时空，就算是府君大人也做不到，更不要说你一个人间天师。”
<br/>　　叶少阳听了这话，道：“你说的没错，我是做不到，而且我也不想穿越啊，但有一样东西，我相信你听过。”
<br/>　　萧逸云立刻问道：“什么？”
<br/>　　“山海印！被徐福从阴司带走的山海印！”
<br/>　　萧逸云当场就懵了，望着叶少阳，许久说道：“徐福把山海印偷走了？”
<br/>　　“你不知道？”叶少阳也是略感诧异，随即想起。当初有关山海印的传闻是徐文长告诉自己的，可能阴司也没几个人知道。于是说道：“你不知道这件事，并不重要，山海印你总是听说过的吧。”
<br/>　　“轮回司的圣物……据说有逆转乾坤、穿越时空的神通……”萧逸云喃喃说道。
<br/>　　叶少阳长出了一口气，总算是说到这一步了，脸上带着一副无奈的表情，望着萧逸云。
<br/>　　萧逸云失神地看着他，虽然关于山海印的传说，他是听过的，不过也只是传说而已，从来没听说过真的有人靠山海印成功穿越的。以萧逸云的认知，根本没法相信这种事情的发生。不过……关于这一切，面前这个家伙说的有鼻子有眼的，最重要的是，他说的关于自己暗恋某个姑娘的事，自己还真的没跟别人说过，连那个姑娘自己也不知道，面前这个自己完全不认识的家伙，怎么可能知道？
<br/>　　“我现在脑子有点乱，你让我想想……”萧逸云喃喃说道。
<br/>　　叶少阳点点头，抱着胳膊，等在一边，并不着急。
<br/>　　萧逸云把叶少阳说的话，从头到尾想了一遍，然后转过身，望着叶少阳说道：“你说你是穿越时空而来，那时你告诉我，你为什么要穿越过来？”
<br/>　　叶少阳听了这话，非常想做一个微信表情里“捂脸”的表情，蹲在地上，呵呵傻笑起来。
<br/>　　萧逸云皱眉道：“你笑什么？”
<br/>　　“我这是笑啊？我这是欲哭无泪！”
<br/>　　叶少阳抹了一下脸，用十分委屈的声音说道：“你问我穿越过来干什么，我实话告诉你，我不知道！！因为我根本就不知道为什么会穿越，不知道那个傻比徐福为什么要用山海印把我弄到这里来！我在自己那个世界，还有很多事情要做，我也不想来这里啊啊啊！”
<br/>(本章完)
