　　对于道风，对于风之谷来说，这场战斗已经结束了，虽然跟自己的初衷不同，不过对道风而言，也不是没有收获，至少确定了李浩然就是自己要找的青牛祖师，至于怎么样说服他跟自己合作，也只有回去再慢慢去思考了。
<br/>　　道风提着山河社稷图，离开了空界，大半个捉鬼联盟的成员，现在都在山河社稷图里，也只有等回到人间之后再把他们放出来了。
<br/>　　对于空界诸门派来说，战斗并没有结束——
<br/>　　恰恰相反，空界势力，与尸族大军的战斗，才刚刚开始。
<br/>　　阴间，天子殿后花园。
<br/>　　一个儒生打扮的男子站在花园一棵老槐树下，出神地望着天边远处。正是崔府君。虽然人在阴司，但是他仿佛听到了来自空界的钟声。
<br/>　　他身后不远处，站着萧逸云，垂手而立，不敢说话。
<br/>　　半晌，崔府君叹了口气，道：“东皇出，天劫现……天劫总算是真正的开启了，我原以为最先应该是在鬼域，没想到却是应在青冥界，你可知我为何要跟空界断绝关系，不让那些修行者离开空界？”
<br/>　　“这……是为了把天劫的范围控制在空界，免得蔓延到人间和鬼域来？”
<br/>　　“不是阻止蔓延，是尽量拖延，好做出更完备的准备，当然，一切都是希望。”
<br/>　　说到这里，崔府君转过脑袋，看了萧逸云一眼，说道：“你一定很纳闷，我知天下人之命，一定提前就算到橙子有难，为什么还要让她去青冥界，是也不是？”
<br/>　　萧逸云只好点了点头。
<br/>　　“我算出她有死劫，并且应在这次行动上，死劫能破不能躲，这个你知道的。”
<br/>　　萧逸云又点了点头。
<br/>　　所谓死劫能破不能躲，意思是说，一个人如果命犯死劫，只能破劫，如果有高人看出，可以帮其破劫，成功与否，有很多因素，如果破了就没事了，但是想靠躲的，肯定是行不通的。
<br/>　　这是天地大道，崔府君有能力违背，但是不可能去违背。
<br/>　　“大衍之道五十，去一而生，变幻无穷。我只算出她有此一劫，却无法帮她躲避，只能任由她去。事后，我正好以此为借口，为空界重新制定规则，却是为大局考虑。”
<br/>　　萧逸云道：“原来大人早就成竹在胸。”
<br/>　　“非也。我只算出她有难，却不知应在谁身上。”崔府君沉默了一会，说道：“我是没有想到，空界那些人如此无理，我若不去，橙子死劫难渡……但我曾立誓不出鬼域，不徇私情，今日却是犯戒了。”
<br/>　　“这个……大人，偶尔一次，也没什么吧？”
<br/>　　萧逸云没有明说，意思是反正您是阴司几位大佬之一，大帝基本上不怎么管事，而且您又是大帝嫡传门生，就算您真犯错，也没人能把您怎么样……
<br/>　　崔府君当然明白他的意思，道：“天子犯法与庶民同罪，我也不能例外，从今日起，百年之内，我将不踏出天子殿一步……”
<br/>　　萧逸云听了这话，心中很是激动，毕竟崔府君前往空界搭救橙子，一半是出于对橙子的宠爱，一般也是因为自己……正不知道说什么好，崔府君道：“你去采东华乔木前来，我亲自为橙子聚魂，数月便能养好，此事不必担心。”
<br/>　　萧逸云连忙谢恩，有崔府君一句话，自然没什么可担心的了。这件事也就放下了，迟疑了一下，问道：“弟子想知道，青冥界两界大战，结局会怎么样？”
<br/>　　“造衅一开，难有宁日，青冥界会乱一阵子，此事自有人关注，无须你我操心。”
<br/>　　“哦，那……关于叶少阳和转世鬼童，不知道阴司会怎么处理？”既然橙子没事，萧逸云又开始关注起叶少阳这个兄弟的情况。
<br/>　　崔府君道：“你去寻找阴阳司那一干阴神，前来阴司投案负罪，届时我从中斡旋，从轻发落便是。转世鬼童也字自有人去追捕，与我天子殿无干，你断不可插手。”
<br/>　　萧逸云只能答应。
<br/>　　“至于叶少阳，则无须费心，生死簿上暂已没他名字了。”
<br/>　　“不会吧！”萧逸云并不知道空界后来发生的事情，一听这话，当场惊呆。
<br/>　　“他并未死，却是去了异界，抹去了一切痕迹。”
<br/>　　萧逸云怔了怔，想到什么，道：“跟徐福有关？”
<br/>　　崔府君没作声，是默认了。
<br/>　　萧逸云傻眼了，半晌说道：“那……他还能回来吗？”
<br/>　　“人在异界，痕迹抹去，连生死簿上都没了信息，自然谁也算不出他的未来，一切，看他自己的造化了。”
<br/>　　萧逸云久久说不出话来。他不知道叶少阳穿越去了什么时代，更无法想象，他在那个时代会有什么样的经历，脑海中浮现出当年在人间偷看的那些穿越剧里的情形，寻秦记之类的，当时觉得很有意思，没想到这种事情居然真切地发生在了叶少阳身上……
<br/>　　萧逸云甚至有些向往，不过他最关心的问题还是，叶少阳究竟还能不能回到这个世界？
