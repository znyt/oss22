　　老程的话让两个天竺籍大唐植物学家一头雾水，互相看了看，又再次看向老程。
<br/>　　却见老程依旧在啃着那条可怜的鱼，一边啃一边说道：“那橡胶树你们种的怎么样了？”
<br/>　　“哦，回大将军，小规模的试种已经完成了，现在……”
<br/>　　“那就是说已经可以种了呗？”程老魔放下手里半死不活的鱼，眼中闪着兴奋的光。
<br/>　　“呃……，可以这么说。”卡皮尔点点头。
<br/>　　“太好了！”老程‘哐’的一拍桌子，双眼放光的对两人说道：“你们两个马上上岸，把需要的种子都带上，跟俺回大唐。”
<br/>　　“啊？！回，回大唐？”卡皮尔一时有点反应不过来。
<br/>　　“对，回大唐，现在万事俱备，只欠东风，就等你们的橡胶树呢。”老程鱼也不吃了，一脸兴奋的说道：“现在带着树种回去，三个月以后就收割，然后运回长安，哈哈……，到时候你们就可以躺着数钱了。”
<br/>　　卡皮尔和黑胖子已经被老程的无知惊呆了，天可怜见，这魔头竟然把橡胶树当成水稻了，还以为回到林邑那边随意的往地里一撒，隔几个月就可以吃了。
<br/>　　无奈之下，卡皮尔只能对老程进行科普，把橡胶是如果从树里割出来的方法讲了一遍。
<br/>　　“这么麻烦？”老程听完卡皮尔的描述之后也有些傻眼。
<br/>　　本来他还想着把卡皮尔带回去，半年时间弄出几万斤橡胶到时候就可以拿这东西回长安将功补过，到了那个时候他就可以顺理成章的回长安了。
<br/>　　可是没想到，这橡胶竟然如此麻烦，光等着树木长成就要好几年，而且一年的产出也并没有多少，并不是像麦子那样随便一割一大片的那种。
<br/>　　于是乎老程郁闷了，抓了半天的脑袋却怎么也想不通为什么橡胶不是那种一割一大片的植物。
<br/>　　卡皮尔并不知道老程的打算，见他郁闷便继续说道：“大将军，那橡胶您还要么？岸上还有一些我们这段时间割出来的，如果您要可以搬到船上来。”
<br/>　　“有多少？”老程再次强打精神问道。
<br/>　　“几百斤。”
<br/>　　“那么少？不要。”老程一听这数量，顿时又没了脾气，翻了个白眼说道：“行了，没你们啥事儿了，该干啥干啥去吧！”
<br/>　　“哎！”卡皮尔一听老程让他们走，乐的跟什么似的，拉着黑胖子逃一般下了船，上岸之后便以最快的速度去的最近的一座城池。
<br/>　　……
<br/>　　木叶城，一座人口只有五千的小城，放在大唐可能连个下县都算不上。
<br/>　　但是在天竺这片土地上，却可以堂而皇之的叫城，而且还有城主。
<br/>　　不过现如今世道变了，城主不吃香了，县令才是这座城的最高管理者，所以木叶城也可以叫木叶县。
<br/>　　卡皮尔进城之后直奔以前的城主府，也就是现在的县衙。
<br/>　　不得不说，与大唐境内的县衙相比，这天竺境内的县衙简直就是天堂一般的存在，此前历代城主费尽心思打理出来的城堡随着一声令下，全都被充公，变成了所谓的县衙。
<br/>　　不得不说，这还真是一种讽刺。
<br/>　　……
<br/>　　“什么？又要调人？”县令丁大人，大头兵出身，大字不识一筐，不过丫杀敌有功，结果就成了县令。
<br/>　　“是的，丁大人，在下也是没有办法，这是太子殿下和尉迟将军的命令。”卡皮尔扯虎皮做大旗，放出李承乾的尉迟宝林的名声来唬人。
<br/>　　没办法，这些大兵头县令根本没办法交流，而且张嘴就是三字经，自己公爵的名号在这山高皇帝远的天竺境内根本没有任何作用。
<br/>　　“来，帮我看看，这上面写的是个啥。”丁大人倒拿着由卡皮尔递上来的手令看了半天，最后交给了身边的一个随从。没办法，谁让他不识字呢。
<br/>　　随从对此也很尴尬，拿过手令看了半天，最后确认的点点头，俯在丁大人身边说道：“大人，是真的。”
<br/>　　“哦，既然是真的，那就按他的要求办，要多少人给他多少人。”丁大人打了个哈欠，摇摇头去后面睡觉了，留下随从和卡皮尔尴尬的视着。
<br/>　　……
<br/>　　不过要说起来人这种生物其实还真挺奇怪的，想当年戒日王统治的时候，治下百姓一个个那是懒的要命，而换了大唐来统治却完全颠倒了。
<br/>　　卡皮尔看着那些挥汗如雨，连饭都不吃的雇工简直不敢相信自己的眼睛。
<br/>　　这还是以前的那些人么？什么时候这帮人这么勤劳了？
<br/>　　黑胖子也很惊讶，瞅着那帮人不要命一样挖矿，栽种，浇水简直就像是看到了外星人。
<br/>　　忙了一整天，时间到了晚上，直到天黑的彻底看不见光了，那些负责栽种的家伙们才拖着疲惫的身体回到各自的家中。
<br/>　　卧槽，这什么情况啊这是？竟然真的连饭都不吃？卡皮尔和黑胖子看着那些准备好的饭食，脑袋都要想炸了也没有想出结果。
<br/>　　于是第二天等到那些雇工再次来到所谓的种植园时，卡皮尔问出了心中的疑惑。
<br/>　　“废话，你要是天天头上悬着一把刀，动不动就砍头，你也这么干活。”忙碌中的雇工把一身工作服的卡皮尔当成了他们自己人，翻了个白眼说道：“以前的好日子现在是别想喽，我劝你最好还是好好干，将来如果能混一份大唐的户籍文书，成为一个真正的唐人，那才叫牛逼。”
<br/>　　啥？啥意思？卡皮尔愣了一下：“哎你给我说说呗！”
<br/>　　“还说什么？你不知道啊？新颁布的法令，现在不管从事任何工作，你只要没有完成当日工作量，如果有人举报了你，就会有人来查，查证属实二话不说，直接就是三十板子，而且还要关小黑屋。”
<br/>　　“而举报的人只要连续十次实名举报，而且全部查验属实，那就会得到大唐的户籍文书，从此成为一个大唐人，然后就不用像我们这样玩儿命的干活了。”
<br/>　　卡皮尔听完之后都有些无语了，也不知道是谁想出这么缺德的主意，心说你这样一来大家伙儿谁还干活啊，天天盯着别人举报不就行了。
<br/>　　不过转念一想又觉得不对，这事儿要连续十次举报，你天天游手好闲的盯着别人，立刻就会有被举报的风险，到时候只怕免不了走上被举报人的道路。
<br/>　　高，高啊！这特么谁想出来的损招！
