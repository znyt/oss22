        从今以后，天涯海角，他都愿意陪她去。
<br/>
<br/>        他一直纠结的那件事情，在此刻也彻底醒悟了。
<br/>
<br/>        她说得没错，明明她就是她，他又为什么要把她分割成两个不同的人？
<br/>
<br/>        以前，他的执着让他错过了太多太多，现在，不应该再错过了才对。
<br/>
<br/>        因为决定要环球旅行，所以得办理各个国家的旅行签证，他们不得不继续在澳洲逗留。
<br/>
<br/>        比起跟秦烈在一起游玩，童筱颖发现自己更喜欢和傅夜沉待在一块。
<br/>
<br/>        虽然，他没有秦烈那么风趣幽默，但是她跟他在一起时，心里的感觉就像是嘴里喊着一颗蜜糖。
<br/>
<br/>        童筱颖感觉自己好像是第一次来这个国家，但是潜意识里又对这个国家很熟悉。
<br/>
<br/>        不知不觉中，童筱颖在一处两层的小洋楼院门外停下了脚步。
<br/>
<br/>        院内杂草横生，像是很久都没人住了一样，不过那棵金合欢树却吸引了童筱颖的注意力。
<br/>
<br/>        现在已经过了花期，草地上还有掉落的金合欢花残骸。
<br/>
<br/>        傅夜沉下意识地看了下门柱上贴着的门牌号，才发现，这里就是筱颖和阿景曾经住过的地方。
<br/>
<br/>        他们在澳洲生活了三年，在筱颖的脑海里，一定有那三年的美好的回忆。
<br/>
<br/>        童筱颖有些出神地望着那棵金合欢树，脑海里闪过一些若隐若现的画面。
<br/>
<br/>        “你若是喜欢这里，我把这里买下来。我们就住在这里。”傅夜沉开口说道。
<br/>
<br/>        童筱颖身心一怔，偏过头去有些难以置信地看向傅夜沉，反问：“真的可以吗？”
<br/>
<br/>        “当然可以！”傅夜沉释然地微笑着。
<br/>
<br/>        因为原户主是傅昼景，所以，傅夜沉托人买下这幢两层楼的小洋房时遇到了些困难。
<br/>
<br/>        不过，一个星期后，这件事情还是搞定了。
<br/>
<br/>        傅昼景起初舍不得卖，但最终还是把它给卖掉了，虽然不知道他是出于什么原因，又愿意卖掉，但是他并不知道，买主是傅夜沉。
<br/>
<br/>        童筱颖跟民宿的老板娘退了民宿的房间，拖着行李，和傅夜沉一起搬进了“新家”。
<br/>
<br/>        他们在搬进来之前，傅夜沉已经找了家政公司将这里打扫干净了。
<br/>
<br/>        院子里不再有杂草，他还派人在金合欢树下安装了一个秋千。
<br/>
<br/>        傅夜沉虽然愿意陪着她在这里住下来，但只是暂时的，并且，他也没把工作的事情给丢掉。
<br/>
<br/>        大概凌晨的时候，童筱颖睡了一觉，从床上爬起来后，觉得有些口渴，于是出门想去厨房喝水，结果无意间看到书房的灯还是亮着的。
<br/>
<br/>        她好奇地走到书房门口，只听到里面传来傅夜沉正在开视频会议的声音。
<br/>
<br/>        最近这几天，他确实是白天陪她玩到尽兴，到了晚上等她睡着了后，他似乎才开始忙自己工作上的事情。
<br/>
<br/>        童筱颖觉得，自家老公这么辛苦，她是不是应该要有些表示？
<br/>
<br/>        开完会后，他一定会饿了吧！
<br/>
<br/>        嗯！
<br/>
<br/>        不如，去给老公做点夜宵吧！
<br/>
<br/>        童筱颖心想着，便自顾自地去了厨房。
<br/>
<br/>        可是，当她站在冰箱前，看着冰箱里的食材时，却一脸迷茫。
<br/>
<br/>        这些天在家里，一日三餐都是傅夜沉做给她吃的，因为她不会下厨。
<br/>
<br/>        “……”童筱颖顿时有些苦恼起来。
<br/>
<br/>        “你肚子饿了吗？想吃什么？”
<br/>
<br/>        不知何时，身后忽然响起了傅夜沉的声音。
<br/>
<br/>        童筱颖循声望去，不假思索地脱口而出：“可乐鸡翅。”
<br/>
<br/>        她只是看到冰箱里有瓶可乐，才随口说的，好在冷冻室里还有新鲜的鸡翅。
<br/>
<br/>        “那我搜个教程，现学现做。”傅夜沉忍俊不禁地说。
<br/>
<br/>        童筱颖连忙将冰箱里的可乐和鸡翅拿了出来，她一边放到中岛台上，一边看着傅夜沉问道：“大哥，你是不是很忙？如果很忙，真的不用陪我。我一个人，也可以的。”
<br/>
<br/>        “不忙，就是偶尔需要开个会，解决一些高管们解决不了的问题。”傅夜沉看着手机上的教程回答道。
<br/>
<br/>        童筱颖便不再说什么了。
<br/>
<br/>        看完教程后，傅夜沉挽起了衣袖，开始处理准备好的食材。
<br/>

<br/>        可乐鸡翅被傅夜沉端到童筱颖的面前时，下一秒，这小女人便毫不犹豫地拿起筷子开吃。

<br/>        鸡翅的肉很嫩又多汁，甜度适中，总之，童筱颖几乎是一口就能干掉一个。

<br/>        一盘可乐鸡翅被童筱颖消灭后，傅夜沉意味深长地问道：“你吃饱了吗？”

<br/>        童筱颖如小鸡琢磨般点了点头：“嗯！吃饱了！”
<br/>
<br/>        “现在是不是该轮到我吃了？”傅夜沉会心一笑。
